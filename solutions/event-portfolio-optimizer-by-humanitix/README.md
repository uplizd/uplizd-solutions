## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your workspace and project to initialize the workflow.
3. Authenticate your Humanitix connection within the Composio Toolset node.
4. Ensure nodes are correctly connected: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives your natural language queries regarding event performance.
*   **Agent**: Processes requests and determines the necessary data retrieval steps.
*   **Composio Toolset**: Executes secure API calls to Humanitix to fetch event and donation metrics.
*   **Chat Output**: Delivers formatted insights, charts, or recommendations back to the user.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
* `Analyze the donation ROI for all events held in the last quarter.`
* `Which event type had the highest ticket conversion rate and donation volume?`
* `Summarize the performance of the 'Spring Gala' and suggest improvements for next year.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as your specialized event data analyst.
*   Maintain a professional, data-focused tone.
*   Prioritize quantitative metrics when comparing event performance.
*   Always cite specific event names and dates when providing recommendations.

### 2) Composio Toolset Node
Requires a valid Humanitix API key. Ensure the connection scope includes read access to events, orders, and donation records.

### 3) Tool Availability
*   **Event Retrieval**: Fetch lists of past and upcoming events.
*   **Attendee Analytics**: Access ticket sales and registration data.
*   **Donation Processing**: Query donation totals linked to specific event IDs.
*   **Reporting Tools**: Generate summary statistics for portfolio-wide performance.
