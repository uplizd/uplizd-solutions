## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select the Gumroad Revenue Optimization template.
2. Click "Import Flow" to initialize the workspace with the pre-configured nodes.
3. Connect your Gumroad API credentials within the Composio Toolset node settings.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives your specific queries regarding sales performance or pricing strategy.
- **Agent**: Processes data using the LLM to synthesize trends and generate strategic recommendations.
- **Composio Toolset**: Executes secure API calls to fetch Gumroad transaction data and product metrics.
- **Chat Output**: Delivers clear, actionable insights and pricing suggestions directly to your interface.

### 3) Run the Flow
Open the Playground and test the assistant with these prompts:
- `Analyze my top 5 products from the last 30 days and suggest a pricing strategy to increase conversion.`
- `Which product bundles have shown the highest revenue growth in the current quarter?`
- `Identify any stalled products that need a marketing push or price adjustment to improve sales velocity.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a specialized revenue analyst.
- Use a high-reasoning model to ensure accurate interpretation of sales data.
- Instruct the agent to prioritize revenue-per-customer metrics.
- Maintain a professional, data-driven tone in all generated recommendations.

### 2) Composio Toolset Node
- Requires a valid Gumroad API key with read-access to your store's transaction and product data.
- Ensure the connection scope includes `products.read` and `sales.read` permissions.

### 3) Tool Availability
- **Gumroad Data Fetcher**: Retrieves real-time transaction history and product metadata.
- **Revenue Calculator**: Computes performance metrics and growth percentages.
- **Trend Analyzer**: Identifies patterns in sales velocity and customer purchase frequency.
