## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" link above to open the solution template.
2. Select your preferred workspace to initialize the workflow environment.
3. Authenticate your Dictionary API credentials within the integration settings.
4. Ensure nodes are correctly connected: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives your raw email draft and context requirements.
- **Agent**: Processes the text, identifies areas for improvement, and queries the dictionary.
- **Composio Toolset**: Executes the lookup and refinement operations using the Dictionary API.
- **Chat Output**: Delivers the polished, professional version of your email.

### 3) Run the Flow
Open the Playground and test the assistant with these prompts:
- `Refine this email draft for a more professional and concise tone: [Paste Draft]`
- `Find a more impactful synonym for 'good' in this context: [Paste Sentence]`
- `Check this email for clarity and suggest improvements for a formal client audience: [Paste Draft]`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a professional editor, prioritizing clarity and tone.
- Use a system prompt that emphasizes "professional, concise, and polite" writing styles.
- Configure the temperature to 0.3 to ensure consistent, predictable, and accurate vocabulary suggestions.
- Enable the agent to ask for clarification if the input email context is insufficient.

### 2) Composio Toolset Node
- Provide your active Dictionary API key within the Composio configuration panel.
- Ensure the connection scope allows for read-only access to dictionary definitions and thesaurus data.

### 3) Tool Availability
- **Definition Lookup**: Retrieves precise meanings for words to ensure accurate usage.
- **Synonym Finder**: Provides context-aware alternatives to enhance vocabulary.
- **Usage Example Generator**: Offers examples to verify if a word fits the professional context.
