## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the PPC Ad Copy Intelligence Agent template from the marketplace.
3. Connect your SEMrush API credentials within the Composio configuration panel.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives your target keywords or competitor domains.
*   **Agent**: Processes the input and orchestrates the research and generation logic.
*   **Composio Toolset**: Executes API calls to fetch competitor data and ad metrics.
*   **Chat Output**: Returns optimized ad copy and strategic recommendations.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
*   `Analyze the top 3 competitors for the keyword "CRM software" and suggest 5 high-CTR ad headlines.`
*   `Compare my current ad copy for "cloud storage" against the market leader and suggest improvements.`
*   `Generate a set of ad descriptions for a holiday sale campaign based on recent competitor trends.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a specialized marketing analyst.
*   Focus on data-driven insights rather than generic creative output.
*   Maintain a professional, conversion-oriented tone in all generated copy.
*   Prioritize actionable recommendations based on the provided SEMrush data.

### 2) Composio Toolset Node
Requires a valid SEMrush API key to retrieve competitor ad data. Ensure the connection scope includes read access to advertising research and domain analytics endpoints.

### 3) Tool Availability
*   **Domain Analytics**: Fetching competitor traffic and keyword data.
*   **Ad Research**: Extracting live ad copy and historical performance metrics.
*   **Content Generator**: Creating optimized ad variations based on research.
