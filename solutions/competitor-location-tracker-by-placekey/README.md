## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and select the Competitor Location Tracker.
2. Click "Import" to add the workflow to your workspace.
3. Connect your required API credentials (Placekey, CRM) in the integration settings.
4. Ensure nodes are correctly linked from Chat Input to Agent, then to the Composio Toolset, and finally to Chat Output.

### 2) Setup the Nodes
*   **Chat Input**: Receives your search parameters (e.g., target city, industry, or competitor name).
*   **Agent**: Processes the request and determines the necessary location data queries.
*   **Composio Toolset**: Executes the API calls to fetch and verify location intelligence.
*   **Chat Output**: Delivers a structured summary of competitor locations and market insights.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
- `Find all competitor locations for 'Coffee Shop' within a 5-mile radius of downtown Chicago.`
- `Compare the market saturation of 'Electronics Retailers' in Austin versus Dallas.`
- `List the most recent competitor location openings in the Pacific Northwest region.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical engine, interpreting location queries and structuring the data for your CRM.
- Focus on identifying geographic patterns and competitor trends.
- Prioritize accuracy in location matching and data normalization.
- Ensure the output is formatted for easy ingestion into downstream business tools.

### 2) Composio Toolset Node
- Provide your Placekey API key to enable high-fidelity location data retrieval.
- Configure the connection scope to allow the agent to read and write to your target CRM or database.

### 3) Tool Availability
- **Placekey API**: For precise location identification and cross-platform data matching.
- **CRM Connectors**: For syncing insights directly into your existing sales or operations records.
- **Geospatial Analysis Tools**: For calculating distances and density metrics.
