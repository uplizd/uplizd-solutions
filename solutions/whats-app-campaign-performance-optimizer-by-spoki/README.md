## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your workspace and import the workflow into your Uplizd dashboard.
3. Connect your Spoki API credentials within the integration settings.
4. Ensure nodes are correctly mapped and all required API keys are active.

### 2) Setup the Nodes
- **Chat Input**: Receives manual requests or scheduled triggers for campaign analysis.
- **Agent**: Processes campaign data, identifies trends, and formulates optimization strategies.
- **Composio Toolset**: Executes API calls to fetch messaging metrics and push updates to Spoki.
- **Chat Output**: Delivers the final performance report and optimization recommendations to the user.

### 3) Run the Flow
Use the Playground to test your campaign analysis:
- `Analyze the performance of the 'Summer Sale' campaign from the last 7 days.`
- `Which WhatsApp templates had the highest click-through rate this month?`
- `Provide optimization suggestions for campaigns with an open rate below 20%.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a specialized Marketing Analyst.
- Focus on identifying statistical outliers in campaign data.
- Maintain a professional, data-driven tone for all generated insights.
- Prioritize actionable recommendations over raw data dumps.

### 2) Composio Toolset Node
- Requires a valid Spoki API key with read/write access to campaign analytics.
- Ensure the connection scope includes `campaigns`, `messages`, and `analytics` permissions.

### 3) Tool Availability
- **Campaign Fetcher**: Retrieves raw performance data for specified date ranges.
- **Template Analyzer**: Compares current template performance against historical averages.
- **Insight Generator**: Synthesizes data into human-readable optimization strategies.
