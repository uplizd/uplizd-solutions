## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the AI Research Assistant template from the library.
3. Connect your Perplexity API key within the integration settings.
4. Ensure nodes are correctly linked from Chat Input to the Agent and finally to the Chat Output.

### 2) Setup the Nodes
*   **Chat Input**: Receives your research topic or specific question.
*   **Agent**: Processes the query and orchestrates the research strategy.
*   **Composio Toolset**: Executes live web searches and data retrieval tasks.
*   **Chat Output**: Delivers the final, cited research report to the user.

### 3) Run the Flow
Open the Playground and test the assistant with these prompts:
*   `"Research the latest trends in generative AI for enterprise sales and provide a summary with citations."`
*   `"Compare the top 3 CRM platforms for small businesses based on recent 2024 reviews."`
*   `"Summarize the impact of recent data privacy regulations on marketing automation tools."`

---

## Configuration
### 1) Language Model (Agent Node)
The agent acts as the primary research strategist.
*   Set the system prompt to prioritize accuracy and source citation.
*   Use a high-reasoning model to ensure complex queries are broken down effectively.
*   Configure the temperature to 0.2 for factual, consistent output.

### 2) Composio Toolset Node
*   Ensure your Perplexity API key is active within the Composio dashboard.
*   Set the connection scope to allow web search and document retrieval permissions.

### 3) Tool Availability
*   **Web Search**: Real-time access to global search engines.
*   **Data Extraction**: Parsing capabilities for summarizing web pages.
*   **Citation Engine**: Formatting tool for valid source attribution.
