# E-commerce Order Processor (Uplizd) - Automated fulfillment and inventory synchronization

## Summary
The E-commerce Order Processor (Uplizd) automates the end-to-end lifecycle of customer orders by bridging Webflow e-commerce data with logistics and inventory systems. This workflow eliminates manual data entry, reduces fulfillment latency, and ensures a single source of truth for stock levels, ultimately driving higher operational efficiency and improved customer satisfaction.

## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to access the solution template.
2. Select your workspace and click "Import Flow" to initialize the workflow in your builder.
3. Authenticate your Webflow and logistics platform connections within the integration settings.
4. Ensure nodes are correctly mapped to your specific Webflow site ID and inventory database schema.

### 2) Setup the Nodes
- **Chat Input**: Receives order event triggers or manual processing requests.
- **Agent**: Analyzes order data, applies business logic, and determines the necessary fulfillment action.
- **Composio Toolset**: Executes API calls to Webflow and logistics providers to update records.
- **Chat Output**: Confirms successful processing or alerts the user to any fulfillment exceptions.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Process the latest pending orders from Webflow and update the inventory database.`
- `Check the status of order #12345 and notify the customer if there is a shipping delay.`
- `Sync current stock levels for all active products in the store.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as the central logic engine for interpreting order data and executing fulfillment rules.
- Instruction: "You are an e-commerce operations assistant. Extract order details from the input and verify inventory availability."
- Instruction: "If an order is valid, trigger the fulfillment process; if stock is low, flag the order for manual review."
- Instruction: "Maintain a professional tone when generating status update messages for customers."

### 2) Composio Toolset Node
- **API Key**: Ensure your Webflow API key and logistics provider credentials are saved in your environment variables.
- **Connection Scope**: Grant read/write access to your store's orders, products, and inventory endpoints.

### 3) Tool Availability
- **Webflow API**: For fetching order details and updating order status fields.
- **Inventory Management Connector**: For real-time stock adjustments and SKU tracking.
- **Notification Service**: For sending automated email or SMS updates to customers.
