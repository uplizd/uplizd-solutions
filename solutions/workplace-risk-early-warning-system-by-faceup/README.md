## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Workplace Risk Early Warning System template from the marketplace.
3. Connect your required API credentials for the integrated communication and monitoring tools.
4. Ensure nodes are correctly linked from **Chat Input** to **Agent**, **Composio Toolset**, and finally **Chat Output**.

### 2) Setup the Nodes
* **Chat Input**: Receives manual triggers or automated system alerts for processing.
* **Agent**: Analyzes the input against defined risk parameters and safety guidelines.
* **Composio Toolset**: Executes data retrieval and alert dispatching across your connected enterprise stack.
* **Chat Output**: Delivers the final risk assessment and recommended mitigation steps to the user.

### 3) Run the Flow
Use the Playground to test the system with these example prompts:
* `Analyze the latest operational logs for any safety protocol deviations.`
* `Check for potential risks in the current project workflow and alert the operations team.`
* `Summarize the top three workplace risks identified this week and suggest mitigation strategies.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical engine for risk detection.
* Use a high-reasoning model (e.g., GPT-4o or Claude 3.5 Sonnet) for accurate pattern recognition.
* System instructions should emphasize safety-first logic and objective risk scoring.
* Configure the temperature to 0.2 to ensure consistent, reliable, and non-hallucinated risk assessments.

### 2) Composio Toolset Node
* Provide your Composio API key to enable secure access to your integrated platforms.
* Ensure the connection scope includes read access to operational logs and write access to notification channels.

### 3) Tool Availability
* **Log Analysis Tools**: For parsing system and operational data.
* **Communication APIs**: For sending alerts via Slack, Email, or Microsoft Teams.
* **Database Connectors**: For retrieving historical incident data to provide context to the Agent.
