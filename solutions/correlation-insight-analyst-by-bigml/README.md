## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and select the Correlation Insight Analyst solution.
2. Click "Import" to add the workflow to your personal workspace.
3. Connect your BigML API credentials and relevant data source integrations via the Composio dashboard.
4. Ensure nodes are correctly mapped: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
*   **Chat Input**: Receives the dataset reference or specific analytical query from the user.
*   **Agent**: Processes the request, formulates the statistical query, and interprets the BigML results.
*   **Composio Toolset**: Executes the data retrieval and triggers the correlation analysis models.
*   **Chat Output**: Delivers the final insights, trend summaries, and actionable recommendations.

### 3) Run the Flow
Use the Playground to test your analysis:
*   `Analyze the correlation between our Q3 marketing spend and regional sales growth.`
*   `Identify the top three variables influencing customer churn in the last six months.`
*   `Run a correlation report on the latest support ticket data to find bottlenecks.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as the analytical bridge, translating natural language queries into machine learning tasks.
*   Maintain a neutral, objective tone when presenting statistical findings.
*   Prioritize clarity by explaining the "why" behind identified correlations.
*   Always cite the confidence level or statistical significance of the findings provided by the model.

### 2) Composio Toolset Node
Requires a valid BigML API key and read-access to your data storage (e.g., S3, Google Sheets, or SQL databases). Ensure the scope allows for data aggregation and model execution.

### 3) Tool Availability
*   **Data Fetching**: Connectors for reading raw CSV, JSON, or database rows.
*   **BigML Analysis**: Capability to trigger correlation, regression, and cluster analysis.
*   **Report Formatting**: Tools to structure findings into bulleted summaries or markdown tables.
