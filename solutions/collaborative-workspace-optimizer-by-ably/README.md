## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and select the Collaborative Workspace Optimizer.
2. Click "Import" to add the workflow to your workspace.
3. Connect your Ably and relevant workspace API credentials in the integration settings.
4. Ensure nodes are correctly mapped to your specific workspace environment and save the configuration.

### 2) Setup the Nodes
*   **Chat Input**: Receives user queries or trigger commands for workspace analysis.
*   **Agent**: Processes activity data, identifies trends, and formulates optimization strategies.
*   **Composio Toolset**: Executes API calls to perform workspace management tasks and data retrieval.
*   **Chat Output**: Delivers clear, actionable reports and confirmation of automated actions to the user.

### 3) Run the Flow
Access the Playground to test the workflow with your live data:
* `Analyze workspace activity for the last 7 days and identify top 3 bottlenecks.`
* `List all inactive channels that should be archived based on current activity.`
* `Generate a productivity report for the engineering team and suggest workflow improvements.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a workspace analyst, prioritizing clarity and operational efficiency.
*   Focus on identifying patterns rather than just raw data points.
*   Maintain a professional, proactive tone when suggesting optimizations.
*   Always provide a clear "Why" behind every recommended workspace change.

### 2) Composio Toolset Node
Requires an active API key with scope permissions for your workspace platform (e.g., Slack, Notion, or custom Ably-integrated apps).

### 3) Tool Availability
*   **Activity Fetchers**: Real-time event log retrieval.
*   **Workspace Management**: Channel/folder creation, archiving, and permission updates.
*   **Analytics Engine**: Pattern recognition and trend reporting tools.
