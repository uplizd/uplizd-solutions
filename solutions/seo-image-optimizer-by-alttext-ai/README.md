## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the solution template.
2. Select your preferred workspace to import the workflow.
3. Authenticate your connected CMS or storage platform via the Composio Toolset.
4. Ensure nodes are correctly connected: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives image URLs or file paths for processing.
- **Agent**: Analyzes the image content and crafts SEO-optimized descriptions.
- **Composio Toolset**: Executes the API calls to update image metadata in your CMS.
- **Chat Output**: Returns a confirmation report of the updated image tags.

### 3) Run the Flow
Use the Playground to test the optimization:
- `Optimize alt text for the image at [URL] using keywords: 'sustainable running shoes'`
- `Scan the latest 10 images in the media library and generate descriptive alt tags`
- `Update alt text for all images in the 'Summer Collection' folder for better SEO`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the SEO strategist and visual interpreter.
- Set the system prompt to prioritize descriptive accuracy and keyword density.
- Configure the model to follow specific brand voice guidelines for alt text.
- Enable structured output to ensure the metadata is formatted for your specific CMS requirements.

### 2) Composio Toolset Node
- Provide your API key for the target CMS (e.g., WordPress, Shopify, or Cloudinary).
- Define the connection scope to allow the agent read/write access to image metadata fields.

### 3) Tool Availability
- **Image Analysis API**: For visual recognition and content extraction.
- **CMS Metadata Manager**: For pushing updates to image alt attributes.
- **SEO Keyword Parser**: For mapping input keywords to generated descriptions.
