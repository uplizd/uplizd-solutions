## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and locate the Content ROI Optimizer.
2. Click "Import" to add the workflow to your workspace.
3. Authenticate your Spotlightr and CRM accounts within the integration settings.
4. Ensure nodes are correctly mapped and all API credentials are saved before triggering the first run.

### 2) Setup the Nodes
* **Chat Input**: Accepts user queries or scheduled triggers for performance analysis.
* **Agent**: Processes video metrics and applies ROI calculation logic.
* **Composio Toolset**: Executes data retrieval and budget adjustment commands across platforms.
* **Chat Output**: Delivers the final ROI report and budget recommendations to the user.

### 3) Run the Flow
Use the Playground to test the workflow with the following prompts:
* `Analyze the performance of all videos in the Q3 campaign and suggest budget shifts.`
* `Which videos have the highest conversion rate for enterprise leads?`
* `Generate a summary of content ROI for the last 30 days.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a marketing analyst, focusing on data synthesis and strategic advice.
* Use a model with strong reasoning capabilities (e.g., GPT-4o or Claude 3.5 Sonnet).
* Instruction: "Prioritize high-conversion metrics over vanity views when calculating ROI."
* Instruction: "Provide clear, bulleted recommendations for budget reallocation."

### 2) Composio Toolset Node
* Ensure the API key is active and the connection scope includes read access to video analytics and write access to ad/budget management platforms.

### 3) Tool Availability
* **Spotlightr API**: For retrieving video engagement and retention data.
* **CRM Connector**: For mapping video views to lead and opportunity records.
* **Analytics/Ad Platform Tools**: For executing budget adjustments and performance reporting.
