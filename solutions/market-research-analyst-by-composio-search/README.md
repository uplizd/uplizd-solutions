## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Market Research Analyst template from the marketplace.
3. Configure your API keys for the search and research tools within the Composio Toolset.
4. Ensure nodes are correctly connected from **Chat Input** to **Agent**, then to **Composio Toolset**, and finally to **Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives the research topic or competitor name from the user.
*   **Agent**: Orchestrates the research logic and synthesizes findings into a coherent report.
*   **Composio Toolset**: Executes web searches and data extraction tasks via connected APIs.
*   **Chat Output**: Delivers the final, formatted market intelligence report to the user.

### 3) Run the Flow
Use the Playground to test your research agent with prompts like:
- `Analyze the current competitive landscape for SaaS project management tools in 2024.`
- `What are the latest trends in AI-driven customer support automation?`
- `Compare the pricing and feature set of [Competitor A] versus [Competitor B].`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical brain of the workflow.
- Use a model with strong reasoning capabilities (e.g., GPT-4o or Claude 3.5 Sonnet).
- Instruct the agent to prioritize verifiable sources and cite them in the final output.
- Set the temperature to 0.2 to ensure factual, concise, and objective reporting.

### 2) Composio Toolset Node
- Provide a valid API key with access to search and web scraping tools.
- Scope the connection to allow the agent to browse and extract data from relevant industry domains.

### 3) Tool Availability
- **Web Search**: For broad information retrieval and news aggregation.
- **Data Extractor**: For parsing specific content from competitor landing pages.
- **Summarizer**: For condensing large volumes of text into executive summaries.
