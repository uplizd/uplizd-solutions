## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Twitter Thread Generator template from the solution library.
3. Connect your preferred LLM and social media integration via the Composio Toolset.
4. Ensure nodes are correctly linked from Chat Input to Agent, then to the Composio Toolset and finally the Chat Output.

### 2) Setup the Nodes
- **Chat Input:** Receives the raw source text or URL to be converted.
- **Agent:** Processes the input, applies brand guidelines, and drafts the thread.
- **Composio Toolset:** Executes the posting or drafting actions on the target social platform.
- **Chat Output:** Displays the final thread preview for manual review before publishing.

### 3) Run the Flow
Use the Playground to test your thread generation with these example prompts:
- `Summarize this article into a 7-tweet thread about AI automation trends: [Insert URL]`
- `Turn these meeting notes into a professional thread about our new product launch: [Insert Text]`
- `Create a thread explaining the benefits of CRM data hygiene using this technical document: [Insert Text]`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as your social media copywriter.
- **Instruction Pattern:**
    - "Act as an expert social media manager specializing in viral thread architecture."
    - "Prioritize clarity, brevity, and high-value takeaways in every tweet."
    - "Maintain a professional yet conversational tone consistent with our brand guidelines."

### 2) Composio Toolset Node
- Provide your API key to authenticate the connection to your social media accounts.
- Set the scope to allow the agent to draft or post content directly to your profile.

### 3) Tool Availability
- **Content Fetcher:** Retrieves text from provided URLs or documents.
- **Social Media Publisher:** Handles the posting or drafting of threads.
- **Formatting Engine:** Ensures the output adheres to character limits and platform-specific syntax.
