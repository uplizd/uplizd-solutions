## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" button to open the solution template.
2. Select your preferred workspace and project destination.
3. Authenticate your required storage and CMS integrations via the connection prompts.
4. Ensure nodes are correctly mapped to your specific API endpoints and folder paths.

### 2) Setup the Nodes
- **Chat Input**: Define the content theme or campaign requirements for the agent.
- **Agent**: Processes the request and applies curation logic to identify the best assets.
- **Composio Toolset**: Executes the search, filter, and move commands across your connected storage platforms.
- **Chat Output**: Provides a summary of the curated assets and their new locations.

### 3) Run the Flow
Use the Playground to test your curation logic:
- `Find and organize all high-resolution images related to 'Summer Campaign 2024' into the 'Approved' folder.`
- `Scan the 'Raw Assets' folder and move any images lacking alt-text to the 'Needs Review' folder.`
- `Select the top 5 images for a LinkedIn post about our new product launch and tag them with 'Q3-Launch'.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as the intelligent filter for your asset library.
- **Instruction Pattern**:
    - "Act as a professional creative director with an eye for high-quality, on-brand visual assets."
    - "Prioritize images that match the provided campaign keywords and technical quality requirements."
    - "Always provide a brief justification for why specific images were selected or flagged."

### 2) Composio Toolset Node
- **API Key**: Ensure your Composio API key is active and authorized for your storage provider.
- **Connection Scope**: Grant read/write access to the specific folders where your raw assets are stored and where curated assets should be delivered.

### 3) Tool Availability
- **File Search**: Capability to scan directory structures for image files.
- **Metadata Management**: Ability to read and write EXIF data and custom tags.
- **File Operations**: Capability to move, copy, or delete assets based on AI-driven decisions.
