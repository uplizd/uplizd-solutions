## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Email Campaign Optimizer JSON template provided in this repository.
3. Connect your preferred email marketing platform via the Composio Toolset.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives the draft email content and campaign parameters from the user.
*   **Agent**: Processes the input, performs sentiment analysis, and generates optimization suggestions.
*   **Composio Toolset**: Executes API calls to fetch campaign metrics or update email drafts in your marketing platform.
*   **Chat Output**: Delivers the optimized content and performance report back to the user.

### 3) Run the Flow
1. `Optimize this draft for a Q3 product launch campaign: [Insert Email Body]`
2. `Generate 5 high-converting subject lines for an abandoned cart email.`
3. `Audit this email content for spam triggers and suggest improvements.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your expert marketing copywriter and deliverability analyst.
*   Adopt a professional, data-driven persona focused on conversion.
*   Prioritize clarity, brevity, and adherence to brand guidelines.
*   Use provided tool outputs to ground all suggestions in performance metrics.

### 2) Composio Toolset Node
*   **API Key**: Ensure your marketing platform (e.g., HubSpot, Mailchimp) is authenticated within the Composio dashboard.
*   **Connection Scope**: Grant read/write access to campaign drafts and analytics reporting endpoints.

### 3) Tool Availability
*   **Content Analyzer**: Sentiment and readability scoring.
*   **Campaign Manager**: Fetching and updating email drafts.
*   **Analytics Connector**: Retrieving historical open and click-through rates.
