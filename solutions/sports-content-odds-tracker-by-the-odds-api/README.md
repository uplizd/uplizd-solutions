## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and select the Sports Content Odds Tracker solution.
2. Click "Import" to add the workflow to your workspace.
3. Connect your API credentials for The Odds API and any required content distribution platforms.
4. Ensure nodes are correctly linked and authenticated before initiating the first run.

### 2) Setup the Nodes
- **Chat Input**: Define the sport, league, or specific event you wish to track.
- **Agent**: Processes the request and interprets the betting data provided by the toolset.
- **Composio Toolset**: Executes the API calls to fetch live odds and market data.
- **Chat Output**: Delivers the finalized, data-backed content ready for publication.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
- `Analyze the current odds for the upcoming Lakers vs. Celtics game and write a short betting preview.`
- `What are the most significant line movements for today's Premier League matches?`
- `Generate a social media post highlighting the best underdog odds for this weekend's UFC event.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical engine that interprets raw odds data.
- Use a model with strong reasoning capabilities to identify trends in betting data.
- Ensure the instruction set emphasizes accuracy and objective reporting.
- Configure the system prompt to maintain a consistent tone suitable for your target audience.

### 2) Composio Toolset Node
- Provide your valid API key for The Odds API within the Composio configuration.
- Set the connection scope to include read-only access to the specific sports leagues you intend to cover.

### 3) Tool Availability
- **Odds Fetcher**: Retrieves live moneyline, spread, and total data.
- **Market Monitor**: Tracks historical line changes for specific events.
- **Data Formatter**: Converts JSON API responses into structured text for the agent.
