## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Campaign Performance Reporter template file.
3. Authenticate your marketing platform connections within the Composio Toolset.
4. Ensure nodes are correctly connected: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives the specific campaign name or date range for the requested report.
*   **Agent**: Processes the request and determines which metrics to query from the marketing API.
*   **Composio Toolset**: Executes the API calls to fetch real-time engagement data.
*   **Chat Output**: Formats the retrieved data into a concise, professional performance report.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
* `Generate a performance summary for the 'Q3 Product Launch' email campaign.`
* `Compare the click-through rates of our last three newsletter campaigns.`
* `Identify the top 3 underperforming segments from the current marketing cycle.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a marketing data analyst.
*   **Instruction Pattern:**
    *   Adopt a professional, data-driven tone for all reports.
    *   Prioritize actionable insights over raw data dumps.
    *   Always cite the specific campaign metrics used for the analysis.

### 2) Composio Toolset Node
*   **API Key:** Ensure your marketing platform API key is stored securely in the Composio environment.
*   **Connection Scope:** Grant read-only access to campaign analytics and reporting endpoints.

### 3) Tool Availability
*   **Campaign Retrieval:** Fetching lists and status of active/past campaigns.
*   **Metric Extraction:** Pulling open, click, bounce, and conversion data.
*   **Data Formatting:** Converting JSON responses into human-readable markdown reports.
