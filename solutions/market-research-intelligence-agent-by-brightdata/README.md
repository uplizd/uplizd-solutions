## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Market Research Intelligence Agent template.
3. Configure your BrightData API credentials within the integration settings.
4. Ensure nodes are correctly connected: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives your research query or target industry topic.
*   **Agent**: Processes the request and orchestrates the search strategy.
*   **Composio Toolset**: Executes the BrightData queries to fetch required web intelligence.
*   **Chat Output**: Delivers the synthesized research report or data summary.

### 3) Run the Flow
Use the Playground to test your agent with specific research prompts:
*   `"Analyze the current pricing strategy of [Competitor Name] in the SaaS sector."`
*   `"What are the top 3 emerging trends in renewable energy for Q3 2024?"`
*   `"Compare the feature sets of [Product A] and [Product B] based on recent public documentation."`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a research lead, prioritizing accuracy and synthesis.
*   Instruct the agent to prioritize primary sources over secondary commentary.
*   Set the temperature to 0.2 for analytical, fact-based output.
*   Define the output format (e.g., Markdown table, bulleted list, or executive summary).

### 2) Composio Toolset Node
*   Provide your BrightData API key to enable web scraping capabilities.
*   Set the connection scope to include search, extraction, and data parsing tools.

### 3) Tool Availability
*   **Web Search**: For broad industry news and current events.
*   **Data Extraction**: For pulling specific tables or pricing data from target URLs.
*   **Summarization Engine**: For condensing long-form articles into key insights.
