## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the YouTube Channel Growth Analytics Agent template from the marketplace.
3. Authenticate your YouTube account within the Composio connection settings.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives user queries regarding channel performance or specific video analytics.
- **Agent**: Processes natural language requests and orchestrates the retrieval of YouTube data.
- **Composio Toolset**: Executes API calls to fetch video stats, subscriber counts, and audience insights.
- **Chat Output**: Delivers formatted reports and strategic recommendations back to the user.

### 3) Run the Flow
Use the Playground to test the agent with prompts like:
- `Analyze the performance of my last 5 videos and suggest improvements for retention.`
- `What was the primary driver of subscriber growth for this channel over the past month?`
- `Compare the engagement metrics of my top 3 videos and summarize the key takeaways.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a specialized data analyst capable of interpreting YouTube API outputs.
- Instruct the agent to prioritize high-impact metrics like retention rate and click-through rate.
- Configure the agent to provide actionable advice rather than just raw data.
- Maintain a professional, data-driven tone suitable for marketing reporting.

### 2) Composio Toolset Node
- Provide your YouTube API credentials within the Composio dashboard.
- Ensure the connection scope includes read-only access to channel analytics and video metadata.

### 3) Tool Availability
- **Video Statistics Fetcher**: Retrieves views, likes, comments, and shares.
- **Audience Insight Engine**: Analyzes retention graphs and demographic data.
- **Channel Growth Monitor**: Tracks subscriber velocity and historical performance trends.
