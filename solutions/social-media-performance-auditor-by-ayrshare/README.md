The Social Media Performance Auditor is an intelligent Uplizd workflow that aggregates engagement data across social channels to provide actionable optimization insights. By leveraging the Composio Toolset to interface with platforms like Ayrshare, this solution enables marketing teams to maintain a single source of truth for content performance, identify high-growth trends, and automate the reporting pipeline to improve overall social media ROI.

## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Social Media Performance Auditor template from the library.
3. Connect your Ayrshare and social media accounts within the integration settings.
4. Ensure nodes are correctly linked from Chat Input to the Agent and finally to the Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives your request for a performance audit or specific platform query.
- **Agent**: Processes the request, interprets the data, and formulates optimization strategies.
- **Composio Toolset**: Executes API calls to fetch metrics and analytics from social platforms.
- **Chat Output**: Delivers the final audit report and strategic recommendations to the user.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Generate a performance summary for my Instagram and LinkedIn accounts for the last 30 days.`
- `Which of my recent posts had the highest engagement and why?`
- `Provide three actionable recommendations to improve my Twitter reach based on last week's data.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a specialized social media analyst.
- Use a high-reasoning model to ensure accurate interpretation of engagement data.
- Instruct the agent to prioritize actionable insights over raw data dumps.
- Configure the system prompt to maintain a professional, data-driven tone.

### 2) Composio Toolset Node
- Provide your Ayrshare API key to enable secure access to your social media analytics.
- Set the connection scope to include read-only access for analytics and reporting endpoints.

### 3) Tool Availability
- **Analytics Fetcher**: Retrieves likes, shares, comments, and reach data.
- **Trend Analyzer**: Processes historical data to identify growth patterns.
- **Report Generator**: Formats findings into clear, structured summaries.
