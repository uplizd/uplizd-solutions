## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" button above to open the solution template.
2. Select your workspace and click "Import" to load the workflow into your dashboard.
3. Authenticate your required CRM and search tool connections within the Composio settings.
4. Ensure nodes are correctly connected: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the target competitor name or market segment for analysis.
- **Agent**: Orchestrates the research process and synthesizes findings into a strategic report.
- **Composio Toolset**: Executes live searches and fetches data from connected CRM and market intelligence platforms.
- **Chat Output**: Delivers the final, formatted competitive analysis report to the user.

### 3) Run the Flow
Open the Playground and test the agent with these prompts:
- `Analyze the current market positioning of [Competitor Name] and identify three key weaknesses.`
- `Compare our pricing model against [Competitor Name] based on the latest public data.`
- `Generate a battle card for our sales team focusing on [Competitor Name]'s recent product launch.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a strategic analyst.
- Use a high-reasoning model (e.g., GPT-4o or Claude 3.5 Sonnet) for accurate data synthesis.
- Instruct the agent to prioritize recent data over historical records.
- Ensure the agent maintains a neutral, objective tone in all generated reports.

### 2) Composio Toolset Node
- Provide your valid API Key to authorize the agent to access external tools.
- Set the connection scope to include your CRM (e.g., OnePage) and web search tools.

### 3) Tool Availability
- **Web Search**: For retrieving real-time news and competitor website updates.
- **CRM Connector**: For cross-referencing competitor data with your internal account records.
- **Data Parser**: For cleaning and structuring unstructured data from the web.
