## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" link above to open the solution in your workspace.
2. Select your preferred environment and click "Import Flow."
3. Connect your required search and analytics accounts via the Composio Toolset.
4. Ensure nodes are correctly mapped: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Define the target country and primary product category for the SEO analysis.
- **Agent**: Processes search data and applies SEO frameworks to generate strategic recommendations.
- **Composio Toolset**: Executes real-time queries across search and analytics APIs to fetch live market data.
- **Chat Output**: Delivers the final strategy report, including keyword lists and content optimization suggestions.

### 3) Run the Flow
Use the Playground to test your strategy:
- `Analyze the search intent for 'SaaS project management' in the German market.`
- `Compare keyword difficulty for 'cloud storage' between the US, UK, and Japan.`
- `Identify top 5 content gaps for our brand in the French market based on current search trends.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as an expert International SEO Consultant.
- Use a model with strong reasoning capabilities (e.g., GPT-4o or Claude 3.5 Sonnet).
- Instruct the agent to prioritize high-volume, low-competition keywords.
- Ensure the agent maintains a consistent tone of voice aligned with your global brand guidelines.

### 2) Composio Toolset Node
- Provide your API keys for the integrated search analytics platforms.
- Configure the connection scope to allow read-only access to search volume and ranking data.

### 3) Tool Availability
- **Search Analytics API**: For fetching real-time keyword performance and volume data.
- **Market Research Connector**: For retrieving regional search trends and competitor metrics.
- **Content Analysis Tool**: For evaluating keyword relevance and intent alignment.
