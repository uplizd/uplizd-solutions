## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the landing page optimization template from the solution library.
3. Connect your preferred web analysis tools via the Composio Toolset.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the URL of the landing page to be analyzed.
- **Agent**: Processes the page content and applies optimization heuristics.
- **Composio Toolset**: Fetches page metadata and external performance signals.
- **Chat Output**: Delivers the final audit report and prioritized improvement plan.

### 3) Run the Flow
Use the Playground to test the workflow with the following prompts:
- `Analyze this landing page [URL] and suggest three changes to improve conversion.`
- `Perform a UX audit on [URL] focusing on mobile readability and CTA placement.`
- `Review the copy on [URL] and suggest more persuasive alternatives for the main hero section.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a specialized CRO consultant.
- **Instruction Pattern:**
    - Focus on data-driven conversion principles and modern UX/UI standards.
    - Prioritize recommendations based on potential impact on user engagement.
    - Maintain a professional, objective, and actionable tone in all reports.

### 2) Composio Toolset Node
- Requires an API key for the connected web analysis or browser-automation tool.
- Ensure the connection scope includes read access to web content and page structure metadata.

### 3) Tool Availability
- **Web Scraper**: Extracts text, layout, and meta-information from the target URL.
- **Analytics Connector**: Pulls performance data (if integrated) to provide context for the audit.
- **Task Manager**: Allows the agent to log identified improvements directly into your project management system.
