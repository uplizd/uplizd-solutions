## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the SEO Keyword Generator template from the library.
3. Connect your preferred LLM and the required Composio Toolset.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
* **Chat Input**: Define the product description, target audience, and primary pain points.
* **Agent**: Processes input data to generate optimized keyword strategies based on search intent.
* **Composio Toolset**: Executes search queries and data lookups to validate keyword relevance.
* **Chat Output**: Delivers a structured list of keywords, search intent labels, and content suggestions.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
* `Generate a list of 10 high-intent keywords for a B2B SaaS platform focused on automated CRM data hygiene.`
* `Identify long-tail search queries for a project management tool targeting remote-first engineering teams.`
* `Create a keyword strategy for a new feature launch that helps sales teams track deal opportunities.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your SEO strategist, synthesizing product data into actionable search insights.
* Set the temperature to 0.7 for a balance of creative keyword discovery and analytical precision.
* Use the system prompt to enforce a focus on "high-intent" and "long-tail" search terms.
* Instruct the agent to categorize all keywords by funnel stage (TOFU, MOFU, BOFU).

### 2) Composio Toolset Node
* Provide your API key to enable the agent to interface with search and analytics tools.
* Configure the connection scope to allow read-only access to search trend APIs.

### 3) Tool Availability
* **Search API**: Retrieves real-time search volume and trend data.
* **Analytics Connector**: Validates keyword performance against existing site data.
* **Content Validator**: Checks keyword density and relevance against provided product documentation.
