## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution in your workspace.
2. Select your preferred workspace and project folder to import the workflow.
3. Configure your API credentials for the LLM and any connected research tools.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
* **Chat Input**: Receives the character name and the source text or book title.
* **Agent**: Analyzes the input, performs literary research, and synthesizes character traits.
* **Composio Toolset**: Connects to external knowledge bases to fetch literary context and archetypal definitions.
* **Chat Output**: Delivers the structured character analysis report to the user.

### 3) Run the Flow
Open the Playground and test with these prompts:
* `Analyze the character arc of Jay Gatsby in The Great Gatsby, focusing on his internal motivations.`
* `Compare the archetypal roles of Hermione Granger and Katniss Everdeen.`
* `Identify the primary personality traits and potential character flaws of Sherlock Holmes.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a literary expert. Use the following instruction pattern:
* Adopt the persona of a literature professor with a specialization in narrative theory.
* Maintain objectivity while identifying both explicit and implicit character traits.
* Cite specific narrative behaviors to support all analytical conclusions.

### 2) Composio Toolset Node
Provide your API key and ensure the connection scope includes access to web search and literary database tools to facilitate deep research.

### 3) Tool Availability
* **Web Search**: For retrieving real-time literary criticism and academic summaries.
* **Knowledge Retrieval**: For accessing definitions of literary archetypes and narrative structures.
* **Data Formatting**: For outputting character profiles in structured JSON or Markdown formats.
