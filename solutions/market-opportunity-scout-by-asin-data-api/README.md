## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to navigate to the solution page.
2. Select "Import Workflow" to add the Market Opportunity Scout to your Uplizd workspace.
3. Configure your API credentials within the environment settings to enable data access.
4. Ensure nodes are connected correctly: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives the target ASINs or category keywords from the user.
*   **Agent**: Processes the request and determines which analytical tools to invoke.
*   **Composio Toolset**: Executes the data retrieval and market intelligence queries.
*   **Chat Output**: Delivers the synthesized market report and strategic recommendations.

### 3) Run the Flow
Use the Playground to test the workflow with the following prompts:
*   `Analyze the market potential for the top 5 ASINs in the 'Ergonomic Office Chair' category.`
*   `What are the common weaknesses in the top-rated products for this ASIN: [Insert ASIN]?`
*   `Compare the pricing and feature set of these three ASINs and suggest a competitive positioning strategy.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical brain, interpreting market data and generating strategic insights.
*   Focus on objective analysis of quantitative data points.
*   Maintain a professional, data-driven tone for all recommendations.
*   Prioritize identifying gaps between current market offerings and consumer demand.

### 2) Composio Toolset Node
Requires a valid API key with access to Amazon-related data connectors. Ensure the connection scope includes read-only access to product listing and review data.

### 3) Tool Availability
*   **ASIN Data Extractor**: Fetches real-time pricing, rank, and metadata.
*   **Review Sentiment Analyzer**: Processes customer feedback to extract qualitative insights.
*   **Competitor Comparison Engine**: Normalizes data across multiple products for side-by-side analysis.
