## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Industry Trend Monitor template from the solution library.
3. Configure your API keys for Perplexity AI and the Composio Toolset.
4. Ensure nodes are correctly connected: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
*   **Chat Input**: Receives the industry sector or specific trend topic to monitor.
*   **Agent**: Processes the query and orchestrates the research workflow.
*   **Composio Toolset**: Executes web searches and data extraction tasks.
*   **Chat Output**: Delivers the synthesized trend report to the user.

### 3) Run the Flow
Use the Playground to test your monitor with these prompts:
* `Analyze the current trends in generative AI for enterprise SaaS for Q3.`
* `What are the top 3 emerging challenges in the renewable energy sector this month?`
* `Summarize recent market shifts in the fintech industry regarding regulatory compliance.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a specialized research analyst.
*   Maintain a neutral, analytical tone focused on data-backed insights.
*   Prioritize recent sources (last 30 days) to ensure trend relevance.
*   Structure output with clear headings: "Market Signal," "Potential Impact," and "Strategic Recommendation."

### 2) Composio Toolset Node
*   Requires a valid Perplexity AI API key.
*   Scope should be set to allow web search and document retrieval capabilities.

### 3) Tool Availability
*   **Web Search**: For fetching the latest news and articles.
*   **Data Summarizer**: For condensing long-form research into executive summaries.
*   **Trend Tracker**: For identifying recurring patterns across multiple search results.
