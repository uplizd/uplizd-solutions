## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your workspace and import the workflow into your Uplizd dashboard.
3. Authenticate your survey and CRM platforms within the Composio connection settings.
4. Ensure nodes are correctly mapped and all API credentials are saved before activating the flow.

### 2) Setup the Nodes
- **Chat Input**: Receives triggers from survey platforms or manual user requests.
- **Agent**: Processes performance data and generates optimization strategies.
- **Composio Toolset**: Executes data retrieval and updates survey configurations.
- **Chat Output**: Delivers actionable insights and optimization reports to your team.

### 3) Run the Flow
Use the Playground to test the workflow with these prompts:
- `Analyze the completion rate for the Q3 Product Feedback survey and suggest three improvements.`
- `Which survey questions have the highest drop-off rate this week?`
- `Create a summary report of survey performance for the last 30 days.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical engine for your survey data.
- Use a high-reasoning model (e.g., GPT-4o or Claude 3.5) for accurate trend interpretation.
- Provide clear instructions on how to prioritize survey drop-off points.
- Ensure the agent has access to the latest survey performance schema.

### 2) Composio Toolset Node
- Provide your API key to enable secure communication between Uplizd and your survey platforms.
- Set the connection scope to allow read/write access to survey metrics and configuration settings.

### 3) Tool Availability
- **Survey API Connector**: For fetching response data and completion metrics.
- **CRM Integration**: For mapping survey data to specific customer profiles.
- **Notification Service**: For sending alerts regarding survey performance dips.
