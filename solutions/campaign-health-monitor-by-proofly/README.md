## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to navigate to the solution page.
2. Select "Import" to add the Campaign Health Monitor workflow to your workspace.
3. Authenticate your required marketing and analytics tool connections via the Composio dashboard.
4. Ensure nodes are correctly mapped and all API credentials are saved to the configuration panel.

### 2) Setup the Nodes
- **Chat Input**: Receives manual queries or scheduled triggers to initiate a campaign health check.
- **Agent**: Processes performance data and applies logic to determine campaign health status.
- **Composio Toolset**: Executes API calls to fetch real-time metrics from your marketing platforms.
- **Chat Output**: Delivers a summary report or specific alert regarding campaign health to your team.

### 3) Run the Flow
Use the Playground to test your monitor with these prompts:
- `Check the performance of the 'Summer Sale' campaign and list any underperforming assets.`
- `Are there any active campaigns with a conversion rate below 2%?`
- `Summarize the impact of social proof widgets on our current landing page traffic.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as your dedicated marketing analyst, focusing on data interpretation and alert prioritization.
- Analyze incoming metrics against predefined performance thresholds.
- Prioritize alerts based on the severity of the performance dip.
- Maintain a professional, data-driven tone for all generated reports.

### 2) Composio Toolset Node
Requires an active API key for your marketing analytics platform. Ensure the connection scope includes read-access to campaign performance and engagement data.

### 3) Tool Availability
- **Analytics API**: For fetching real-time traffic and conversion data.
- **CRM Connector**: For syncing campaign status with lead and opportunity records.
- **Notification Service**: For dispatching alerts via email or Slack.
