## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and select the Multi-Platform Banner Adapter.
2. Click "Import" to add the workflow to your workspace.
3. Connect your AdRapid and design platform credentials via the Composio integration menu.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives the source image URL and target platform specifications.
*   **Agent**: Interprets design requirements and triggers the appropriate transformation logic.
*   **Composio Toolset**: Executes the API calls to AdRapid for image processing and asset delivery.
*   **Chat Output**: Returns the download links for the newly generated, platform-optimized banners.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Resize this master banner for LinkedIn, Facebook, and Instagram feed dimensions.`
- `Take this source image and generate a 300x250 display ad and a 728x90 leaderboard banner.`
- `Adapt the provided creative for a mobile story format and export as a high-quality PNG.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as the orchestrator for design logic and API execution.
- Use a model capable of high-precision instruction following for image metadata.
- Ensure the system prompt defines the hierarchy of target platforms and aspect ratios.
- Maintain a strict output format for the Composio Toolset to ensure successful API calls.

### 2) Composio Toolset Node
- Provide your AdRapid API key to enable secure access to the creative engine.
- Set the connection scope to allow read/write access to your asset library.

### 3) Tool Availability
- **Image Processing**: Handles cropping, scaling, and format conversion.
- **Asset Management**: Manages file retrieval from cloud storage and delivery of final outputs.
- **Platform Metadata**: Provides the library of standard dimensions for major ad networks.
