# Community Development Advisor (Uplizd) - Data-driven insights for urban planning and community growth

## Summary
The Community Development Advisor is an intelligent Uplizd workflow that synthesizes Census Bureau data to provide actionable insights for urban planning, economic development, and community resource allocation. By automating the retrieval and analysis of complex demographic and geographic datasets, this solution enables planners and policy-makers to make evidence-based decisions, accelerating the pipeline from raw data to community impact.

## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Community Development Advisor template from the solution library.
3. Connect your required API credentials for the Census Bureau and any supporting data tools.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives your specific query regarding a region or demographic metric.
*   **Agent**: Interprets the request and orchestrates the data retrieval strategy.
*   **Composio Toolset**: Executes the specific API calls to fetch and process census data.
*   **Chat Output**: Delivers the synthesized analysis and recommendations directly to the user.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
* `Analyze the population growth trends in the downtown district over the last five years.`
* `Identify neighborhoods with the highest need for new educational resources based on current demographic data.`
* `Generate a summary report on housing density and income levels for the target development zone.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a specialized urban planning consultant.
*   Maintain an objective, analytical tone focused on data accuracy.
*   Prioritize the synthesis of multiple data points over simple retrieval.
*   Provide clear, actionable recommendations based on the analyzed statistics.

### 2) Composio Toolset Node
*   Ensure your Census Bureau API key is active and configured within the Composio dashboard.
*   Define the scope of access to allow the agent to query specific demographic and geographic endpoints.

### 3) Tool Availability
*   **Census Data Fetcher**: Retrieves demographic, economic, and geographic datasets.
*   **Statistical Analyzer**: Performs calculations on retrieved data to identify trends.
*   **Report Formatter**: Structures findings into professional, policy-ready documentation.
