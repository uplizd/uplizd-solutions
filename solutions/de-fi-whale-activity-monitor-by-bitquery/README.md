## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd solution library and locate the **DeFi Whale Activity Monitor**.
2. Click the "Import" button to add the workflow to your workspace.
3. Authenticate your Bitquery and communication tool credentials within the connection settings.
4. Ensure nodes are correctly linked from the **Chat Input** through the **Agent** and **Composio Toolset** to the **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives your monitoring parameters and alert preferences.
- **Agent**: Processes blockchain data and filters for high-value whale activity.
- **Composio Toolset**: Executes queries against the Bitquery API to fetch real-time transaction data.
- **Chat Output**: Delivers formatted alerts and insights to your chosen destination.

### 3) Run the Flow
Use the Playground to test your monitoring logic with these prompts:
- `Track all transactions over $1M involving the ETH/USDC liquidity pool today.`
- `Identify the top 5 most active wallets interacting with decentralized lending protocols in the last hour.`
- `Alert me whenever a wallet with a balance over $5M interacts with a new smart contract.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical engine that interprets raw blockchain data into human-readable insights.
- Prioritize accuracy in transaction value parsing and wallet identification.
- Maintain a neutral, data-focused tone for all generated alerts.
- Focus on identifying anomalies that deviate from standard wallet behavior.

### 2) Composio Toolset Node
- Ensure your Bitquery API key is active and has sufficient credits for real-time data streaming.
- Configure the connection scope to allow read-only access to blockchain transaction logs.

### 3) Tool Availability
- **Bitquery API**: Provides deep-dive access to on-chain transaction data and wallet history.
- **Notification Connectors**: Enables delivery of alerts via email, Slack, or webhook.
- **Data Formatting Tools**: Cleans and structures raw JSON blockchain data into readable reports.
