## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Campaign Inspiration Generator template from the solution library.
3. Connect your Adyntel API credentials within the integration settings.
4. Ensure nodes are correctly linked: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives your target industry, product focus, and campaign goals.
- **Agent**: Processes input and queries the Adyntel toolset for relevant ad trends.
- **Composio Toolset**: Executes precise API calls to fetch competitive ad intelligence.
- **Chat Output**: Delivers structured creative briefs and campaign inspiration directly to the user.

### 3) Run the Flow
Open the Playground and try these prompts:
- `Generate 3 creative campaign concepts for a B2B SaaS project management tool targeting remote teams.`
- `Analyze current ad trends for fitness apps and suggest a hook for our upcoming summer sale.`
- `Create a campaign brief based on the top-performing ads in the digital marketing education space.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a creative strategist and market analyst.
- Use a model capable of high-level creative synthesis (e.g., GPT-4o or Claude 3.5 Sonnet).
- Instruction: Focus on actionable creative outputs rather than generic summaries.
- Instruction: Maintain a professional, results-oriented tone suitable for marketing briefs.

### 2) Composio Toolset Node
- Provide your Adyntel API key to enable secure access to ad intelligence data.
- Ensure the connection scope includes read access to ad performance and trend analytics.

### 3) Tool Availability
- **Adyntel Trend Search**: Fetches real-time industry ad performance data.
- **Creative Synthesis Engine**: Processes trend data into structured campaign briefs.
- **Data Formatter**: Converts raw API responses into readable, actionable marketing copy.
