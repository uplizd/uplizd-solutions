## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your workspace and click "Import" to load the workflow into your builder.
3. Authenticate your required search and analytics integrations within the Composio dashboard.
4. Ensure nodes are correctly connected: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the target keyword or URL to be optimized.
- **Agent**: Processes search data and formulates optimization recommendations.
- **Composio Toolset**: Executes live search queries and fetches analytics data.
- **Chat Output**: Delivers the final SEO report and actionable content changes.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Analyze the SERP for "best CRM software" and suggest 5 content improvements for my landing page.`
- `Find the keyword gap between my article at [URL] and the top 3 ranking competitors.`
- `Generate an optimized meta-title and description for a post targeting "AI workflow automation".`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as an SEO analyst, synthesizing search data into clear, actionable tasks.
- Use a model with strong reasoning capabilities (e.g., GPT-4o or Claude 3.5 Sonnet).
- Instruct the agent to prioritize high-volume, low-competition keywords.
- Maintain a professional, data-driven tone in all content recommendations.

### 2) Composio Toolset Node
- Provide your API key to enable secure access to search and analytics connectors.
- Set the connection scope to allow read-only access to your web analytics and search console data.

### 3) Tool Availability
- **Search API**: For real-time SERP data retrieval.
- **Analytics Connector**: For tracking existing page performance metrics.
- **Content Parser**: For analyzing existing page structure and keyword density.
