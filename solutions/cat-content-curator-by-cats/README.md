## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Cat Content Curator template using the provided solution link.
3. Configure your API credentials within the Composio Toolset node.
4. Ensure nodes are correctly connected: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the topic or content parameters from the user.
- **Agent**: Analyzes the request and determines the best search strategy for content.
- **Composio Toolset**: Executes the API calls to fetch and filter the requested media.
- **Chat Output**: Delivers the curated content links and metadata back to the user.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Find 5 trending cat videos from today for a social media post.`
- `Curate a list of high-engagement cat images suitable for a blog header.`
- `Search for viral cat content related to 'funny cats' and format it for Instagram.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the content strategist, interpreting user intent and managing the search logic.
- Instruct the agent to prioritize content with high engagement metrics.
- Define specific content quality filters (e.g., resolution, relevance, sentiment).
- Ensure the agent formats the final output as a ready-to-use social media draft.

### 2) Composio Toolset Node
- Provide your API keys for the relevant social media platforms (e.g., YouTube, Instagram, Reddit).
- Set the connection scope to "Read-Only" to ensure secure content retrieval.

### 3) Tool Availability
- **Search API**: Capability to query social platforms for trending media.
- **Filter Engine**: Capability to process and rank content based on engagement data.
- **Formatter**: Capability to convert raw API data into structured social media posts.
