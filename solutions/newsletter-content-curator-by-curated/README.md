## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select the "Newsletter Content Curator" template.
2. Click "Import" to initialize the workflow in your workspace.
3. Connect your required API credentials (e.g., RSS, CMS, or Email tools) via the Composio Toolset.
4. Ensure nodes are correctly linked from **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the topic or industry focus for the current curation cycle.
- **Agent**: Analyzes the input and executes the search, filtering, and summarization logic.
- **Composio Toolset**: Connects to external platforms to fetch content and push curated drafts.
- **Chat Output**: Delivers the final curated list or confirms the draft has been pushed to your CMS.

### 3) Run the Flow
Use the Playground to test your curation logic:
- `Find and summarize the top 5 articles on AI trends from the last 24 hours.`
- `Curate a list of industry news for our weekly marketing newsletter.`
- `Search for recent updates on content marketing and draft a summary for the team.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your editorial assistant.
- Focus on tone consistency and brevity.
- Prioritize factual accuracy and source attribution.
- Configure the agent to follow a specific editorial style guide.

### 2) Composio Toolset Node
- Provide your API keys for the content sources and target publishing platforms.
- Set the connection scope to allow the agent to read from your sources and write to your drafts.

### 3) Tool Availability
- **RSS/Web Scrapers**: For real-time content discovery.
- **CMS Connectors**: For pushing curated content to your website or newsletter platform.
- **Summarization Engines**: For condensing long-form articles into newsletter-ready snippets.
