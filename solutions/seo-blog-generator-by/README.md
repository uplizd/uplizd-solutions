## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to access the solution template.
2. Select your preferred workspace and project destination.
3. Configure your environment variables for the required LLM and API connections.
4. Ensure nodes are correctly wired: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Accepts the target topic, primary keywords, and desired tone.
- **Agent**: Processes the input to draft the content structure and introduction.
- **Composio Toolset**: Fetches relevant research data or pushes drafts to your CMS.
- **Chat Output**: Displays the final blog outline and introduction for review.

### 3) Run the Flow
Open the Playground and test the flow with these prompts:
- `Generate a blog outline for "The Future of AI in CRM" targeting a professional B2B audience.`
- `Create an introduction for a blog post about "Remote Work Productivity" using a casual and encouraging tone.`
- `Draft a 5-section outline for a guide on "Automating Sales Pipelines" including keywords: CRM, automation, and lead scoring.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your expert content editor.
- Set the system prompt to define the brand voice and SEO constraints.
- Use a high-reasoning model (e.g., GPT-4o or Claude 3.5) for better structural coherence.
- Enable temperature settings between 0.7 and 0.8 for creative yet focused output.

### 2) Composio Toolset Node
- Provide your API key to enable connection to your preferred CMS or research tools.
- Set the connection scope to allow the agent to read/write to your content repository.

### 3) Tool Availability
- **Content Research**: Tools for gathering current trends and search intent data.
- **CMS Connectors**: Capabilities for pushing finalized outlines to platforms like WordPress or Webflow.
- **SEO Analysis**: Tools to validate keyword density and header structure.
