## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Import Workflow."
2. Upload the `social-proof-competitive-analyzer` JSON configuration file.
3. Connect your required API credentials for the integrated marketing and research tools.
4. Ensure nodes are correctly mapped: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
* **Chat Input**: Receives your query regarding specific competitors or industry segments.
* **Agent**: Orchestrates the research, sentiment analysis, and competitive comparison logic.
* **Composio Toolset**: Executes data retrieval from your connected marketing and CRM platforms.
* **Chat Output**: Delivers a structured report summarizing competitive gaps and optimization opportunities.

### 3) Run the Flow
Use the Playground to test the workflow with these prompts:
* `Compare our current case study volume with our top 3 competitors in the SaaS space.`
* `Analyze the sentiment of recent reviews for [Competitor Name] and identify our key differentiators.`
* `Generate a list of recommended social proof assets we should prioritize based on current market trends.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a strategic marketing analyst.
* **Recommended instruction pattern:**
    * Focus on objective data comparison rather than subjective opinion.
    * Prioritize actionable insights that can be implemented within a 30-day marketing cycle.
    * Maintain a professional, data-driven tone suitable for executive reporting.

### 2) Composio Toolset Node
* **API Key:** Ensure your Composio API key is active and authorized for the necessary data connectors.
* **Connection Scope:** Grant read-only access to your connected CRM and marketing analytics platforms to ensure data integrity.

### 3) Tool Availability
* **Market Research Tools:** For gathering external competitor data.
* **CRM Connectors:** For internal performance benchmarking.
* **Sentiment Analysis Engine:** For processing qualitative review data.
* **Reporting Utilities:** For formatting the final output into actionable summaries.
