## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the template in your workspace.
2. Connect your Instagram account via the integration settings.
3. Configure your notification channels (e.g., Slack or Email) to receive the AI-generated alerts.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the trigger signal or manual request to initiate a brand scan.
- **Agent**: Processes the raw Instagram data, performs sentiment analysis, and determines the appropriate action.
- **Composio Toolset**: Executes API calls to fetch Instagram data and push updates to external platforms.
- **Chat Output**: Delivers the final summary report or notification to your dashboard or team communication channel.

### 3) Run the Flow
Use the Playground to test your monitor with these prompts:
- `Scan recent mentions for the last 24 hours and summarize the overall sentiment.`
- `Identify any urgent negative comments that require immediate attention from the support team.`
- `Create a report of the top 5 most engaged posts from the last week.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your social media analyst, interpreting unstructured social data into actionable insights.
- Instruct the agent to prioritize mentions from verified accounts or high-follower profiles.
- Define specific keywords or hashtags that should trigger high-priority alerts.
- Set the tone for automated responses to ensure consistency with your brand voice.

### 2) Composio Toolset Node
- Provide your Instagram API credentials and ensure the necessary read/write scopes are enabled.
- Connect your secondary tools (e.g., CRM, Slack, or Trello) to the toolset to enable automated downstream actions.

### 3) Tool Availability
- **Instagram API**: For fetching posts, comments, and mention data.
- **Sentiment Analysis Tool**: For classifying user feedback.
- **Notification Service**: For sending alerts to team communication platforms.
- **CRM Connector**: For logging interactions against existing customer profiles.
