## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Sustainability Marketing Storyteller template from the library.
3. Connect your required API credentials in the integration settings.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the raw data or project ID for the sustainability report.
- **Agent**: Processes the data, applies brand voice, and drafts the marketing narrative.
- **Composio Toolset**: Connects to your data sources to fetch the latest environmental metrics.
- **Chat Output**: Delivers the final, formatted marketing copy for review and distribution.

### 3) Run the Flow
Use the Playground to test your storytelling agent with the following prompts:
- `Generate a LinkedIn post summarizing our Q3 carbon offset achievements in the Amazon rainforest.`
- `Write a 200-word blog intro explaining why our latest project in Kenya is critical for local biodiversity.`
- `Create a professional summary of our current carbon footprint reduction for the upcoming annual CSR report.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your expert sustainability copywriter.
- Use a high-reasoning model (e.g., GPT-4o or Claude 3.5 Sonnet) for accurate data interpretation.
- Set the system prompt to prioritize clarity, transparency, and brand-specific tone.
- Define a "No Greenwashing" policy in the instructions to ensure all claims are factually supported by the input data.

### 2) Composio Toolset Node
- Provide your API key to enable secure access to your sustainability and CRM platforms.
- Configure the connection scope to allow read-only access to your carbon offset databases.

### 3) Tool Availability
- **Data Fetcher**: Retrieves real-time metrics from your carbon offset provider.
- **Content Formatter**: Applies templates for social media and long-form articles.
- **Brand Voice Validator**: Checks generated content against your saved brand guidelines.
