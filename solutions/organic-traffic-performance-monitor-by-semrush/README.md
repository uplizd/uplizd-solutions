## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the solution template.
2. Select your workspace and project destination.
3. Authenticate your Semrush account within the Composio Toolset.
4. Ensure nodes are correctly connected: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives your request for domain analysis or specific keyword reports.
- **Agent**: Processes the intent and determines which Semrush metrics to fetch.
- **Composio Toolset**: Executes the API calls to retrieve real-time search data.
- **Chat Output**: Delivers the summarized SEO performance report to your interface.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
- `Analyze the organic traffic performance for domain.com over the last 30 days.`
- `What are the top 5 keyword ranking changes for my site this week?`
- `Compare the organic search visibility of my domain against competitor.com.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your SEO analyst, interpreting complex search data into human-readable insights.
- Instruct the agent to prioritize significant ranking changes (±5 positions).
- Configure the agent to provide context on why a specific metric matters for growth.
- Ensure the agent maintains a professional, data-first tone in all summaries.

### 2) Composio Toolset Node
- Provide your Semrush API key within the Composio dashboard.
- Set the connection scope to include read-only access for analytics and domain reporting.

### 3) Tool Availability
- **Domain Analytics**: Fetch traffic volume, keyword counts, and authority scores.
- **Keyword Research**: Retrieve ranking positions and search volume data.
- **Competitor Analysis**: Compare performance metrics across multiple domains.
