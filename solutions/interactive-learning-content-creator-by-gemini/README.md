## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to navigate to the solution page.
2. Select "Import Flow" to add the Interactive Learning Content Creator to your workspace.
3. Connect your preferred LLM provider and the necessary Composio Toolset integrations.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives the raw curriculum text or syllabus document.
*   **Agent**: Processes the input, applies pedagogical frameworks, and instructs the toolset.
*   **Composio Toolset**: Executes external API calls to generate and format multimedia assets.
*   **Chat Output**: Delivers the finalized, interactive content ready for distribution.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
* `Create a 5-question interactive quiz based on the provided curriculum on Python basics.`
* `Generate a visual outline for a multimedia module explaining cloud computing architecture.`
* `Convert the attached text document into a structured lesson plan with embedded interactive prompts.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as an instructional expert. Recommended instructions:
* Act as an expert instructional designer with a focus on active learning.
* Maintain a professional, encouraging, and clear tone suitable for educational content.
* Prioritize accessibility and clarity in all generated multimedia descriptions.

### 2) Composio Toolset Node
* Requires an active API key for the integrated media and document generation services.
* Connection scope should include read/write access to your designated content repository or LMS API.

### 3) Tool Availability
* **Document Parsers**: For extracting text from PDFs and Word documents.
* **Media Generators**: For creating diagrams, charts, and visual assets.
* **Assessment Engines**: For generating quizzes, flashcards, and interactive polls.
