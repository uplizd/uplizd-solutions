## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd Solutions library and select the Currency Trend Analysis Agent.
2. Click "Import" to load the workflow into your private workspace.
3. Configure your Fixer API credentials within the environment variables.
4. Ensure nodes are correctly connected: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives user queries regarding specific currency pairs or timeframes.
- **Agent**: Processes market data and applies analytical logic to interpret trends.
- **Composio Toolset**: Executes secure API calls to fetch external currency data.
- **Chat Output**: Delivers clear, actionable insights and trend summaries to the user.

### 3) Run the Flow
Use the Playground to test the agent with the following prompts:
- `What is the current trend for EUR to USD over the last 30 days?`
- `Analyze the volatility of the Japanese Yen against the British Pound for this quarter.`
- `Based on current trends, when is the best time this week to convert funds to AUD?`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a financial data interpreter, focusing on accuracy and trend identification.
- Use a high-reasoning model to ensure complex financial data is synthesized correctly.
- Instruct the agent to prioritize recent data points while acknowledging historical context.
- Maintain a professional, objective tone suitable for financial reporting.

### 2) Composio Toolset Node
- Requires a valid Fixer API key configured within the Composio dashboard.
- Connection scope should be limited to "Read-Only" access for market data retrieval.

### 3) Tool Availability
- **Fixer Exchange Rate Fetcher**: Retrieves live and historical currency conversion data.
- **Data Aggregator**: Combines multiple data points for trend analysis.
- **Notification Dispatcher**: Sends findings to integrated communication channels.
