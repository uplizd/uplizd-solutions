## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and select the Grant Proposal Research Assistant.
2. Click "Import" to load the workflow into your workspace.
3. Authenticate your Semantic Scholar API credentials within the Composio Toolset node.
4. Ensure nodes are correctly mapped from Chat Input to Agent, Toolset, and Chat Output.

### 2) Setup the Nodes
*   **Chat Input**: Receives your research topic or grant proposal objective.
*   **Agent**: Processes the query and orchestrates the search strategy.
*   **Composio Toolset**: Executes precise API calls to Semantic Scholar for data retrieval.
*   **Chat Output**: Delivers the synthesized research summary and gap analysis.

### 3) Run the Flow
Open the Playground and test with these prompts:
*   `"Find the top 5 recent papers on CRISPR gene editing and summarize the current research gaps."`
*   `"Identify key literature supporting the efficacy of carbon capture technologies for a climate grant proposal."`
*   `"Search for recent studies on neural network interpretability and extract the methodology for my literature review."`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a specialized research assistant focused on academic rigor and synthesis.
*   Prioritize recent, highly-cited publications in search results.
*   Maintain a neutral, objective tone suitable for grant writing.
*   Clearly distinguish between established facts and identified research gaps.

### 2) Composio Toolset Node
Requires a valid Semantic Scholar API key. Ensure the connection scope includes read access to academic paper metadata and citation networks.

### 3) Tool Availability
*   **Search API**: Keyword and semantic-based paper retrieval.
*   **Paper Details**: Fetching abstracts, authors, and publication dates.
*   **Citation Graph**: Mapping relationships between foundational and recent research.
