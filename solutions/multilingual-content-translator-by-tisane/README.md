## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Multilingual Content Translator template from the marketplace.
3. Connect your Tisane API credentials within the integration settings.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
* **Chat Input**: Receives the source text and target language parameters.
* **Agent**: Orchestrates the translation logic and enforces brand tone guidelines.
* **Composio Toolset**: Executes the specific translation calls via the Tisane API.
* **Chat Output**: Delivers the finalized, localized content back to the user or destination system.

### 3) Run the Flow
Open the Playground and test with these prompts:
* `Translate the following marketing email into Spanish, ensuring the tone remains professional yet inviting: [Paste Text]`
* `Localize this product feature announcement for a German audience, keeping the technical specifications accurate: [Paste Text]`
* `Convert this social media caption into French and Japanese, optimizing for engagement in each respective culture: [Paste Text]`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as the linguistic bridge, ensuring the output matches your brand's specific voice.
* Use a high-reasoning model to ensure cultural nuance is captured.
* Set system instructions to prioritize idiomatic accuracy over literal translation.
* Define a "Brand Voice" variable to guide the agent on preferred terminology and tone.

### 2) Composio Toolset Node
* Requires a valid Tisane API key to authenticate translation requests.
* Configure the connection scope to allow the agent to access the specific translation endpoints required for your target languages.

### 3) Tool Availability
* **Text Analysis Engine**: Identifies sentiment, intent, and cultural context.
* **Translation Service**: Performs high-fidelity language conversion.
* **Language Detection**: Automatically identifies the source language if not specified.
