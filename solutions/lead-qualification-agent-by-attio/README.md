## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" button above to open the solution template.
2. Select your workspace and project to import the workflow.
3. Connect your Attio account within the Composio Toolset configuration.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the raw lead data or trigger event.
- **Agent**: Analyzes the lead data against your qualification criteria.
- **Composio Toolset**: Executes the API calls to update or enrich records in Attio.
- **Chat Output**: Returns the qualification summary and enrichment status to the user.

### 3) Run the Flow
Open the Playground and test the agent with these prompts:
- `Qualify this lead: { "name": "Jane Doe", "email": "jane@example.com", "company": "TechCorp" }`
- `Enrich the record for Acme Corp in Attio and check if they meet our ICP.`
- `Run a qualification check on the latest inbound lead and update the status to 'Qualified' if score > 80.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as your digital sales assistant, interpreting raw lead data and mapping it to CRM fields.
- Use a model with strong reasoning capabilities (e.g., GPT-4o).
- Instruct the agent to prioritize accuracy in data mapping.
- Define clear thresholds for what constitutes a "qualified" lead.

### 2) Composio Toolset Node
- Provide your Attio API key to establish a secure connection.
- Set the connection scope to allow read/write access to lead and company objects.

### 3) Tool Availability
- `attio_search_records`: Find existing companies or people.
- `attio_update_record`: Update lead status or custom fields.
- `attio_create_record`: Create new entries for net-new leads.
- `data_enrichment_tool`: Fetch external firmographic data.
