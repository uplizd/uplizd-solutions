## Quick Start

### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" button above to open the solution template.
2. Authenticate your Semrush account within the Composio Toolset node.
3. Configure your target domain and niche parameters in the Agent node.
4. Ensure nodes are correctly connected: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Accepts your target niche, competitor domains, or specific search queries.
- **Agent**: Processes search data, performs gap analysis, and generates strategic recommendations.
- **Composio Toolset**: Executes API calls to Semrush to fetch live keyword and ranking data.
- **Chat Output**: Returns a structured report of actionable keyword opportunities and content suggestions.

### 3) Run the Flow
Use the Playground to test your workflow with prompts like:
- `Find 10 high-volume, low-difficulty keywords for the 'SaaS project management' niche.`
- `Compare my domain 'example.com' against 'competitor.com' and list the top 5 content gaps.`
- `Generate a content cluster strategy based on the top 20 trending keywords in the 'AI automation' sector.`

## Configuration

### 1) Language Model (Agent Node)
The agent acts as an SEO consultant, synthesizing complex search data into clear, actionable advice.
- Instruction: Focus on high-conversion, low-difficulty keywords.
- Instruction: Prioritize keywords that align with the user's specified content goals.
- Instruction: Always provide a rationale for why a specific keyword is an "opportunity."

### 2) Composio Toolset Node
- Requires a valid Semrush API key configured within the Composio dashboard.
- Connection scope should include read-only access to domain analytics and keyword research endpoints.

### 3) Tool Availability
- **Keyword Overview**: Fetch volume, CPC, and difficulty metrics.
- **Domain Overview**: Analyze competitor ranking profiles and traffic sources.
- **Organic Research**: Identify specific pages ranking for target keywords.
