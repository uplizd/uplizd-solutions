## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and locate the Cash Flow Forecaster solution.
2. Click "Import" to add the workflow template to your workspace.
3. Authenticate your financial data sources (e.g., Brex, QuickBooks) within the integration settings.
4. Ensure nodes are correctly mapped and the Agent is connected to the Composio Toolset.

### 2) Setup the Nodes
- **Chat Input**: Receives user queries regarding cash status or forecast requests.
- **Agent**: Processes financial data and applies logic to generate predictive insights.
- **Composio Toolset**: Executes secure API calls to fetch transaction and account data.
- **Chat Output**: Delivers clear, summarized financial analysis and recommendations to the user.

### 3) Run the Flow
Use the Playground to test your forecasting capabilities with these prompts:
- `Analyze my cash flow for the next 30 days based on current burn rates.`
- `Identify any anomalies in our spending patterns from the last week.`
- `Generate a summary of our current liquidity position across all connected accounts.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a financial analyst, interpreting complex data sets into plain language.
- Use a high-reasoning model (e.g., GPT-4o or Claude 3.5 Sonnet) for accurate financial calculations.
- Set the system prompt to prioritize data accuracy and conservative forecasting.
- Ensure the agent is instructed to cite specific account sources when providing figures.

### 2) Composio Toolset Node
- Provide your API key within the Composio configuration panel to enable secure data access.
- Set the connection scope to "Read-Only" for financial accounts to ensure data integrity and security.

### 3) Tool Availability
- **Transaction Fetcher**: Retrieves raw ledger entries and categorized spending.
- **Account Balancer**: Pulls real-time balances from linked banking and credit platforms.
- **Report Generator**: Formats raw data into structured tables and executive summaries.
