## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Connect your Agenty account within the Composio Toolset node.
3. Configure your target brand keywords and monitoring frequency in the Agent node.
4. Ensure nodes are correctly linked: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the brand name or specific campaign keywords to track.
- **Agent**: Processes incoming data, performs sentiment analysis, and formats the report.
- **Composio Toolset**: Executes the Agenty search queries to fetch live web and social mentions.
- **Chat Output**: Delivers the summarized sentiment report and mention highlights to the user.

### 3) Run the Flow
Use the Playground to test your monitoring setup:
- `Analyze the sentiment of recent mentions for [Brand Name] over the last 24 hours.`
- `Find all negative mentions of [Brand Name] and summarize the primary customer complaints.`
- `Generate a weekly report comparing mention volume for [Brand Name] against [Competitor Name].`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your reputation analyst. Recommended instructions:
- Maintain an objective tone when categorizing sentiment.
- Prioritize mentions from high-authority domains and social platforms.
- Summarize findings into actionable bullet points for the marketing team.

### 2) Composio Toolset Node
- **API Key**: Input your Agenty API key to enable data extraction.
- **Connection Scope**: Ensure the tool has read access to the web scraping and search endpoints required for your monitoring targets.

### 3) Tool Availability
- **Agenty Web Scraper**: For deep-diving into specific review sites or news portals.
- **Social Search Connector**: For broad-spectrum monitoring across public social media feeds.
- **Sentiment Classifier**: For real-time linguistic analysis of retrieved text.
