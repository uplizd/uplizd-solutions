## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Corporate Giving Coordinator template via the provided solution URL.
3. Authenticate your required API connections (e.g., Daffy, HRIS, or CRM) within the Composio Toolset.
4. Ensure nodes are correctly mapped: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives natural language queries or triggers from your communication platform.
- **Agent**: Processes requests, validates donation data, and determines the appropriate action.
- **Composio Toolset**: Executes API calls to external giving and management platforms to perform updates.
- **Chat Output**: Delivers confirmation messages, status updates, or generated reports back to the user.

### 3) Run the Flow
Open the Uplizd Playground and test the workflow with these prompts:
- `Summarize the total corporate giving impact for Q3 by department.`
- `Approve the pending donation request for employee ID 8842.`
- `Send a reminder to the marketing team about the upcoming matching gift deadline.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as the central coordinator for all CSR-related data operations.
- Use a model with high reasoning capabilities to handle complex approval logic.
- Instruct the agent to prioritize data accuracy and adherence to corporate giving policies.
- Configure the system prompt to maintain a professional and encouraging tone for employee interactions.

### 2) Composio Toolset Node
- Provide your API keys for the specific giving platforms (e.g., Daffy) and internal tools.
- Set the connection scope to allow read/write access for donation records and employee profiles.

### 3) Tool Availability
- **Donation API**: For fetching and updating gift records.
- **HRIS Connector**: For verifying employee eligibility and department data.
- **Notification Service**: For sending automated updates to employees and managers.
- **Reporting Tool**: For aggregating and formatting impact data.
