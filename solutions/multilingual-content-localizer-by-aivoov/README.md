## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and locate the Multilingual Content Localizer.
2. Click "Import" to add the workflow template to your workspace.
3. Connect your Aivoov and relevant CRM or storage accounts via the Composio Toolset.
4. Ensure nodes are correctly linked from **Chat Input** to **Agent**, then to **Composio Toolset**, and finally to **Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives the source text and target language requirements from the user.
*   **Agent**: Orchestrates the translation and voiceover synthesis logic based on the provided input.
*   **Composio Toolset**: Executes the API calls to Aivoov for audio generation and file management.
*   **Chat Output**: Delivers the final localized audio file or download link to the user.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Translate this product announcement into Spanish and generate a professional voiceover.`
- `Create a French voiceover for the following script: [Insert Script Here].`
- `Convert this marketing copy into Japanese and provide the audio file link.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the linguistic and operational brain of the workflow.
- **Recommended instruction pattern:**
    - Act as a professional localization expert proficient in 150+ languages.
    - Prioritize natural phrasing and cultural accuracy when translating input text.
    - Ensure the output is formatted correctly for the Aivoov synthesis engine.

### 2) Composio Toolset Node
- Provide your Aivoov API key within the Composio configuration panel.
- Set the connection scope to allow the agent to read source files and write generated audio assets.

### 3) Tool Availability
- **Aivoov API**: Handles text-to-speech synthesis and language processing.
- **File Management**: Enables the agent to save and retrieve generated audio files.
- **Notification Service**: Alerts the user once the localization process is complete.
