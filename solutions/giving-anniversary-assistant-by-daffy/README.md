## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" button above to open the solution template.
2. Connect your preferred CRM and email service provider via the Composio Toolset.
3. Configure your notification preferences and donor data source mappings.
4. Ensure nodes are correctly connected: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives donor data triggers or manual requests for anniversary outreach.
- **Agent**: Processes donor history to draft personalized, empathetic, and impact-driven messages.
- **Composio Toolset**: Executes CRM lookups, updates donor records, and triggers email delivery.
- **Chat Output**: Confirms successful outreach or logs errors for manual review.

### 3) Run the Flow
Use the Playground to test your workflow with these example prompts:
- `Check for donor anniversaries occurring in the next 7 days and draft personalized emails.`
- `Identify donors who reached their 1-year giving milestone today and update their CRM status.`
- `Generate a summary report of all anniversary outreach sent this month.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a donor relations expert, focusing on empathy and clarity.
- Maintain a warm, appreciative, and professional tone in all communications.
- Reference specific donor history and impact metrics provided by the CRM.
- Ensure all calls to action are clear, respectful, and non-intrusive.

### 2) Composio Toolset Node
- Provide your API keys for your CRM (e.g., Salesforce, HubSpot) and email provider.
- Set the connection scope to allow read/write access to donor contacts and email sending capabilities.

### 3) Tool Availability
- **CRM Connector**: For fetching donor contact details and anniversary dates.
- **Email Service**: For sending personalized outreach messages.
- **Data Logger**: For updating donor engagement history in the CRM.
