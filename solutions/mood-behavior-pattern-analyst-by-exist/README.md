## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Mood & Behavior Pattern Analyst template from the solution library.
3. Connect your preferred data sources within the Composio Toolset configuration.
4. Ensure nodes are correctly linked from Chat Input to Agent, Toolset, and finally Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives your daily logs, mood scores, and activity data.
- **Agent**: Processes the input, applies psychological analysis models, and identifies patterns.
- **Composio Toolset**: Fetches historical data or external calendar/health metrics to enrich the analysis.
- **Chat Output**: Delivers a structured summary of insights, identified triggers, and actionable recommendations.

### 3) Run the Flow
Access the Playground to test your analysis with the following prompts:
- `Analyze my mood logs from this week and identify the top three triggers for my stress.`
- `Based on my activity data, what is the correlation between my exercise routine and my focus levels?`
- `Summarize my behavioral patterns for the last month and suggest one habit change to improve my morning mood.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a behavioral analyst, synthesizing inputs into empathetic and logical insights.
- Use a model with high reasoning capabilities (e.g., GPT-4o or Claude 3.5 Sonnet).
- Instruction: "Act as a professional behavioral analyst. Focus on identifying causal links between user-provided activities and reported emotional states."
- Maintain a supportive, objective, and analytical tone in all generated reports.

### 2) Composio Toolset Node
- Provide your API key within the Composio node settings to enable secure data retrieval.
- Ensure the connection scope includes read-access to your calendar, task management, or health tracking platforms.

### 3) Tool Availability
- **Calendar API**: To correlate scheduled events with stress or mood spikes.
- **Health/Fitness API**: To pull sleep, heart rate, or activity data for physiological context.
- **Note/Journaling API**: To parse unstructured text logs for sentiment and behavioral keywords.
