## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Video Performance Analyst template file.
3. Connect your Spotlightr or video platform API credentials within the integration settings.
4. Ensure nodes are correctly linked: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
*   **Chat Input**: Receives your natural language query regarding video performance.
*   **Agent**: Processes data requests and formulates analytical queries using the LLM.
*   **Composio Toolset**: Executes API calls to fetch video analytics and engagement data.
*   **Chat Output**: Delivers the final performance report and optimization recommendations to the user.

### 3) Run the Flow
Use the Playground to test your workflow with these example prompts:
*   `"Analyze the engagement drop-off for my latest product launch video and suggest three improvements."`
*   `"Which of my top 5 videos had the highest click-through rate last month?"`
*   `"Generate a summary report of total views and average watch time for the Q3 marketing campaign."`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your data analyst, interpreting complex metrics into actionable strategy.
*   Focus on identifying trends rather than just reporting raw numbers.
*   Maintain a professional, data-driven tone in all generated recommendations.
*   Prioritize insights that directly impact conversion and audience retention.

### 2) Composio Toolset Node
Requires an active API connection to your video hosting provider. Ensure the connection scope includes read access to analytics, video metadata, and engagement logs.

### 3) Tool Availability
*   **Analytics Fetcher**: Retrieves view counts, watch time, and retention data.
*   **Metadata Manager**: Accesses video titles, descriptions, and publishing timestamps.
*   **Engagement Tracker**: Monitors comments, likes, and click-through events.
