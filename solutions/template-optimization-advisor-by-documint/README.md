## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Template Optimization Advisor JSON configuration.
3. Connect your Documint and relevant data source credentials in the integration settings.
4. Ensure nodes are correctly linked from Chat Input to Agent, then to the Composio Toolset, and finally to Chat Output.

### 2) Setup the Nodes
*   **Chat Input**: Receives the specific template ID or document category for analysis.
*   **Agent**: Evaluates the template performance and generates optimization recommendations.
*   **Composio Toolset**: Executes the necessary API calls to Documint to fetch or update template data.
*   **Chat Output**: Delivers the summary of findings and proposed improvements to the user.

### 3) Run the Flow
Use the Uplizd Playground to test your templates:
*   `Analyze the performance of the Q3 Sales Proposal template and suggest 3 improvements.`
*   `Check for outdated pricing fields in all active service agreement templates.`
*   `Compare the conversion metrics of the current onboarding document against the previous version.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a specialized document strategist.
*   Focus on identifying structural inefficiencies and content clarity.
*   Prioritize data-backed suggestions over subjective design preferences.
*   Maintain a professional, actionable tone in all generated reports.

### 2) Composio Toolset Node
*   **API Key**: Provide your Documint API key within the Composio configuration.
*   **Connection Scope**: Ensure the agent has read/write access to your template library to perform audits and updates.

### 3) Tool Availability
*   **Documint Fetch**: Retrieve current template schemas and metadata.
*   **Documint Update**: Apply suggested structural or content modifications.
*   **Analytics Connector**: Pull performance data to inform optimization logic.
