## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your workspace and project destination.
3. Authenticate your TikTok account within the connection manager.
4. Ensure nodes are correctly mapped and the API credentials are saved for the Composio Toolset.

### 2) Setup the Nodes
* **Chat Input**: Receives your natural language queries regarding account performance.
* **Agent**: Processes requests and determines which analytics data to fetch.
* **Composio Toolset**: Executes secure API calls to retrieve TikTok metrics.
* **Chat Output**: Delivers formatted insights and strategic recommendations back to the user.

### 3) Run the Flow
Use the Playground to test your agent with prompts like:
* `Analyze my top 5 videos from the last month and tell me why they performed well.`
* `What is the average engagement rate for my recent educational content?`
* `Suggest three content themes based on my highest-performing videos from last week.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a specialized social media analyst.
* Instruct the agent to prioritize engagement rate over raw view counts.
* Define a clear output structure for reports (e.g., Summary, Key Findings, Recommendations).
* Ensure the agent maintains a professional, data-driven tone.

### 2) Composio Toolset Node
* Provide your TikTok API credentials within the Composio dashboard.
* Ensure the connection scope includes read-access for analytics and video metadata.

### 3) Tool Availability
* `get_video_metrics`: Fetches detailed engagement data for specific video IDs.
* `list_recent_posts`: Retrieves a list of recent uploads and their aggregate performance.
* `analyze_engagement_trends`: Aggregates data over time to identify growth patterns.
