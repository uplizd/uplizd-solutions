## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Project Template Optimization Agent template file.
3. Connect your project management platform credentials within the Composio Toolset node.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives triggers or manual requests to analyze specific project templates.
*   **Agent**: Processes delivery data and determines necessary template refinements based on system instructions.
*   **Composio Toolset**: Executes read/write operations on your project management software.
*   **Chat Output**: Confirms successful template updates and provides a summary of changes made.

### 3) Run the Flow
Use the Playground to test the agent with these prompts:
* `Analyze the last 10 completed projects and suggest improvements for the 'Client Onboarding' template.`
* `Update the 'Standard Implementation' template by adding the task 'Final QA Review' based on recent success patterns.`
* `Identify bottlenecks in the current 'Software Development' template and propose a revised task order.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a strategic operations analyst.
*   Focus on identifying patterns that correlate with reduced project duration.
*   Prioritize task sequences that have been marked as 'completed' in successful projects.
*   Maintain a professional, data-driven tone when suggesting template modifications.

### 2) Composio Toolset Node
Requires an active API connection to your project management tool (e.g., Jira, Asana, or ClickUp). Ensure the connection scope includes read access to project history and write access to template configurations.

### 3) Tool Availability
*   **Project History Fetcher**: Retrieves completion data and task timestamps.
*   **Template Editor**: Allows the agent to modify task lists, due dates, and dependencies.
*   **Analytics Engine**: Processes delivery metrics to determine optimization logic.
