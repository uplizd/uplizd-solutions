    - Reduces the time spent on manual data reconciliation by syncing project-level removal data with central reporting dashboards.
- **Carbon Credit Auditor**
    - Provides real-time visibility into the lifecycle of carbon credits, from initial capture to final verification and retirement.

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a sustainability data analyst.
- Use a model with strong reasoning capabilities (e.g., GPT-4o or Claude 3.5 Sonnet).
- Instruction: Focus on data accuracy, adherence to ESG reporting standards, and clear, professional tone.
- Instruction: Always cross-reference data points with the provided toolset before finalizing any report metrics.

### 2) Composio Toolset Node
- Provide your Composio API key to enable secure connections to your sustainability platforms.
- Set the connection scope to allow read-only access to project data and reporting APIs.

### 3) Tool Availability
- **Data Fetcher**: Retrieves raw carbon removal metrics from connected platforms.
- **Registry Validator**: Checks credit status against public or private carbon registries.
- **Report Formatter**: Converts JSON/CSV data into structured text or document formats.
