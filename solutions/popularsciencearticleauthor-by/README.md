## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Popular Science Article Author template from the solution library.
3. Connect your preferred LLM and the necessary Composio Toolset integrations.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
* **Chat Input**: Receives the research topic, URL, or raw text input from the user.
* **Agent**: Analyzes the input, performs research, and drafts the article based on defined style guidelines.
* **Composio Toolset**: Executes search queries and data extraction to gather necessary scientific context.
* **Chat Output**: Delivers the finalized, polished article draft back to the user interface.

### 3) Run the Flow
Use the Playground to test the workflow with prompts such as:
* `Summarize the latest findings on CRISPR gene editing from the provided research paper URL.`
* `Write a 500-word popular science article explaining the impact of quantum computing on modern cryptography.`
* `Draft an engaging blog post about recent breakthroughs in renewable energy storage for a general audience.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a specialized science editor.
* Use a high-reasoning model (e.g., GPT-4o or Claude 3.5 Sonnet) for accurate synthesis.
* Set the system prompt to prioritize clarity, accuracy, and an engaging narrative tone.
* Configure the temperature to 0.7 to balance creativity with factual adherence.

### 2) Composio Toolset Node
* Provide a valid API key to enable access to search and research tools.
* Define the connection scope to include web search and academic database APIs.

### 3) Tool Availability
* **Web Search**: For fetching the latest news and supplementary context.
* **PDF/Document Parser**: For extracting text from uploaded research papers.
* **Citation Manager**: For automatically formatting references in the final output.
