## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Roleplay Scenario Generator template from the marketplace.
3. Connect your preferred LLM provider and the Composio Toolset.
4. Ensure nodes are correctly linked: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives your core character concept or narrative theme.
- **Agent**: Processes the input, applies creative constraints, and generates the scenario.
- **Composio Toolset**: Connects to external storage or creative APIs to save your generated content.
- **Chat Output**: Displays the final, formatted narrative scenario for your review.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Generate a gritty cyberpunk detective character with a secret fear of technology.`
- `Create a fantasy scenario where a merchant and a disgraced knight must escape a collapsing dungeon.`
- `Write a dialogue-heavy scene between two rival space pilots forced to share a life pod.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the creative engine, interpreting your inputs to build coherent narratives.
- Set the system prompt to prioritize "Creative, immersive, and character-driven storytelling."
- Use a temperature setting between 0.7 and 0.9 for high creativity.
- Ensure the model has access to the full conversation history to maintain narrative continuity.

### 2) Composio Toolset Node
- Provide your Composio API key to enable integration with external tools.
- Configure the connection scope to allow the agent to read/write to your preferred document storage or creative database.

### 3) Tool Availability
- **Narrative Database**: For storing character profiles and world lore.
- **Document Manager**: For exporting scenarios to text files or cloud documents.
- **Search API**: For fetching real-world inspiration or historical context to ground your fiction.
