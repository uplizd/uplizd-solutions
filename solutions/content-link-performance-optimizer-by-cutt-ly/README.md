## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your workspace and click "Import" to load the workflow into your builder.
3. Connect your Cutt.ly API credentials within the Composio Toolset node.
4. Ensure nodes are correctly linked: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Accepts user queries regarding specific links or campaign performance timeframes.
- **Agent**: Processes the request, interprets performance data, and generates strategic recommendations.
- **Composio Toolset**: Executes API calls to Cutt.ly to fetch real-time link analytics.
- **Chat Output**: Delivers a human-readable summary of link performance and actionable content suggestions.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
- `Analyze the performance of my top 5 links from the last 7 days and suggest content improvements.`
- `Which referral source has driven the most clicks for my recent product launch campaign?`
- `Summarize the geographic distribution of clicks for the latest newsletter link.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a marketing analyst that translates raw data into strategic insights.
- Instruct the agent to prioritize high-CTR links when making recommendations.
- Ensure the agent maintains a professional, data-focused tone.
- Configure the agent to highlight anomalies or significant drops in engagement.

### 2) Composio Toolset Node
- Provide your Cutt.ly API key in the configuration settings.
- Set the connection scope to allow read-only access to your link analytics data.

### 3) Tool Availability
- **Link Analytics Fetcher**: Retrieves click counts, referral data, and geographic metadata.
- **Trend Aggregator**: Calculates growth percentages and identifies performance patterns.
- **Report Generator**: Formats data into structured summaries for the Chat Output node.
