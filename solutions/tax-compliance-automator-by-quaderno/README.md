## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your preferred workspace to import the workflow.
3. Authenticate your Quaderno and CRM accounts within the integration settings.
4. Ensure nodes are correctly connected and API keys are verified before triggering the first run.

### 2) Setup the Nodes
*   **Chat Input**: Receives transaction details or invoice requests from the user.
*   **Agent**: Processes the request, identifies the required tax logic, and prepares the data.
*   **Composio Toolset**: Executes the API calls to Quaderno for tax calculation and invoice generation.
*   **Chat Output**: Returns the generated invoice link or tax summary to the user.

### 3) Run the Flow
Use the Playground to test the workflow with these prompts:
* `Calculate tax for a $500 software sale to a customer in France.`
* `Generate a compliant invoice for customer ID 98765 with current tax settings.`
* `Verify the tax compliance status for our recent transactions in the European region.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as the financial logic controller, translating user intent into precise API actions.
*   Prioritize accuracy in tax rate identification.
*   Maintain a professional, audit-focused tone in all outputs.
*   Ensure all required fields for invoice generation are extracted from the input.

### 2) Composio Toolset Node
Requires a valid Quaderno API key and appropriate scope permissions to read transaction data and create invoices.

### 3) Tool Availability
*   **Tax Calculation Engine**: Real-time rate lookups.
*   **Invoice Generator**: PDF and digital receipt creation.
*   **Compliance Validator**: Tax ID and jurisdiction verification tools.
