## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your preferred workspace and project to initialize the workflow.
3. Authenticate your SerpApi account within the Composio Toolset configuration.
4. Ensure nodes are correctly connected: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives your search queries or competitor names.
- **Agent**: Processes intent and orchestrates the search strategy.
- **Composio Toolset**: Executes real-time web queries via SerpApi.
- **Chat Output**: Delivers the synthesized intelligence report to your dashboard.

### 3) Run the Flow
Use the Playground to test the agent with these prompts:
- `Analyze the latest product launch from [Competitor Name] and summarize the key features.`
- `What is the current market sentiment regarding the recent release of [Product Name]?`
- `Compare the feature set of our latest release against the top 3 search results for [Industry Keyword].`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a market analyst, transforming raw search data into strategic insights.
- Instruct the agent to prioritize recent, high-authority sources.
- Set the tone to professional, objective, and analytical.
- Configure the agent to highlight discrepancies between competitor claims and user reviews.

### 2) Composio Toolset Node
- Provide your **SerpApi API Key** to enable search capabilities.
- Set the connection scope to allow the agent to perform broad web searches and specific site-based queries.

### 3) Tool Availability
- **Search Engine Querying**: Fetching real-time results for specific keywords.
- **Content Summarization**: Condensing long-form articles and press releases.
- **Sentiment Scoring**: Evaluating positive/negative feedback patterns.
