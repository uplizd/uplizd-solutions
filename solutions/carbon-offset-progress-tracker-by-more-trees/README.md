## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Carbon Offset Progress Tracker template from the marketplace.
3. Connect your required environmental data APIs via the Composio Toolset.
4. Ensure nodes are correctly mapped and all API credentials are authenticated.

### 2) Setup the Nodes
- **Chat Input**: Receives queries regarding project status or report generation requests.
- **Agent**: Processes environmental data and performs logic-based analysis on offset progress.
- **Composio Toolset**: Executes secure API calls to fetch and update external sustainability data.
- **Chat Output**: Delivers formatted status updates, impact summaries, or generated reports.

### 3) Run the Flow
Use the Playground to test your workflow with these example prompts:
- `Generate a summary report of all carbon offset projects for Q3.`
- `Check the status of the reforestation project and update the progress field.`
- `List all carbon credits retired this month and verify their certification status.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your sustainability analyst, interpreting complex environmental data into clear, actionable insights.
- Instruct the agent to prioritize verified data sources for all reporting tasks.
- Configure the agent to maintain a professional, data-driven tone for ESG documentation.
- Set strict output constraints to ensure all reports follow your organization's internal naming conventions.

### 2) Composio Toolset Node
- Provide your API keys for the relevant environmental data providers.
- Limit the connection scope to read/write access only for the specific projects being tracked.

### 3) Tool Availability
- **Data Retrieval Tools**: Fetch project metrics, credit retirement logs, and certification status.
- **Formatting Tools**: Convert raw environmental data into standardized report formats.
- **Update Tools**: Push verified status updates back to your primary project management or CRM system.
