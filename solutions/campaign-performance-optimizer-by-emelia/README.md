## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" button above to open the solution template.
2. Select your preferred workspace to import the workflow configuration.
3. Authenticate your marketing and CRM accounts within the integration settings.
4. Ensure nodes are correctly mapped to your active API credentials and verify all connection statuses.

### 2) Setup the Nodes
* **Chat Input**: Receives performance queries or manual trigger commands from the user.
* **Agent**: Processes engagement data and determines the necessary optimization actions.
* **Composio Toolset**: Executes API calls to update campaign settings or fetch real-time analytics.
* **Chat Output**: Delivers a summary of actions taken and current campaign status updates.

### 3) Run the Flow
Access the Playground in the Uplizd builder to test your workflow with these prompts:
* `Analyze the performance of the latest Q3 email campaign and suggest budget reallocations.`
* `Identify all campaigns with a conversion rate below 2% and pause them.`
* `Generate a summary report of top-performing ad sets from the last 7 days.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical engine, interpreting performance data and executing logic.
* Use a model with strong reasoning capabilities (e.g., GPT-4o or Claude 3.5 Sonnet).
* Instruction pattern: Focus on data-driven decision-making, maintain a professional marketing tone, and prioritize ROI-positive actions.
* Ensure the agent has access to the full context of campaign history to prevent redundant adjustments.

### 2) Composio Toolset Node
* Requires a valid API key for the relevant marketing platforms (e.g., Google Ads, Meta, HubSpot).
* Connection scope should include read/write access to campaign management and reporting endpoints.

### 3) Tool Availability
* **Analytics Fetchers**: Retrieve real-time click-through rates, conversion data, and spend metrics.
* **Campaign Controllers**: Update status, budget, and targeting parameters.
* **CRM Connectors**: Sync engagement data to lead profiles for unified tracking.
