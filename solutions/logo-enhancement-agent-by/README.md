## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Logo Enhancement Agent template file.
3. Connect your preferred Vision Language Model and image storage credentials.
4. Ensure nodes are correctly mapped from Chat Input to Agent to Composio Toolset to Chat Output.

### 2) Setup the Nodes
* **Chat Input**: Receives the image file and enhancement instructions from the user.
* **Agent**: Analyzes the visual data and determines the necessary transformation steps.
* **Composio Toolset**: Executes the image processing tasks and interacts with design APIs.
* **Chat Output**: Delivers the enhanced image file and a summary of the quality improvements made.

### 3) Run the Flow
Use the Playground to test your agent with these prompts:
* `Analyze this logo and tell me if it meets our 4K resolution standards.`
* `Recreate this logo with a transparent background and higher contrast for dark mode.`
* `Evaluate this brand asset and suggest three variations for a social media profile picture.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a visual expert, focusing on brand guidelines and technical image specifications.
* Set the system prompt to prioritize brand consistency and high-fidelity output.
* Enable vision capabilities to allow the model to "see" and interpret the uploaded image files.
* Configure the temperature to a low setting to ensure precise, reproducible design adjustments.

### 2) Composio Toolset Node
* Provide your API key for the relevant design or storage integration.
* Define the connection scope to allow the agent to read from and write to your asset management folders.

### 3) Tool Availability
* **Image Analysis Tools**: For resolution, color profile, and format detection.
* **Generative Design Tools**: For recreating or modifying visual elements.
* **File Management Tools**: For saving and retrieving assets from cloud storage.
