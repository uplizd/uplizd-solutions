## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to navigate to the solution page.
2. Select "Import" to add the pre-configured workflow to your Uplizd workspace.
3. Connect your required API keys (More Trees and any relevant CRM) within the integration settings.
4. Ensure nodes are correctly linked: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives the client identifier and reporting timeframe.
- **Agent**: Processes the request, retrieves data, and drafts the impact narrative.
- **Composio Toolset**: Executes secure API calls to fetch verified sustainability data.
- **Chat Output**: Delivers the finalized, formatted sustainability report to the user.

### 3) Run the Flow
Access the Playground to test the workflow with these prompts:
- `Generate a sustainability impact report for Client ID 8842 for the Q3 period.`
- `Create a summary of total trees planted and carbon offset for all active accounts.`
- `Draft a personalized impact email for our top-tier client highlighting their annual environmental contribution.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical engine that interprets raw data into human-readable insights.
- Instruct the agent to maintain a professional, transparent tone.
- Configure the agent to prioritize verified data points from the More Trees API.
- Set the output format to match your organization’s preferred reporting template.

### 2) Composio Toolset Node
- Provide your API keys for the More Trees platform within the Composio configuration.
- Define the scope to allow read-only access to impact metrics to ensure data security.

### 3) Tool Availability
- **Data Retrieval**: Fetching carbon offset logs and tree-planting counts.
- **Formatting**: Structuring data into professional PDF or text-based reports.
- **Integration**: Syncing generated reports with CRM tools for client record-keeping.
