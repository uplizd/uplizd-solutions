## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Visual Content Discovery Agent template from the marketplace.
3. Connect your required media and social platform accounts via the Composio integration settings.
4. Ensure nodes are correctly linked from **Chat Input** to **Agent**, then to **Composio Toolset**, and finally to **Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives your search parameters, industry keywords, or trend categories.
*   **Agent**: Processes the request, interprets visual intent, and orchestrates the discovery logic.
*   **Composio Toolset**: Executes API calls to external visual platforms and media databases.
*   **Chat Output**: Delivers the curated list of discovered assets and trend insights to your workspace.

### 3) Run the Flow
Use the Playground to test the agent with prompts such as:
- `Find the latest trending visual styles for sustainable fashion brands on Instagram.`
- `Search for high-engagement minimalist photography assets related to tech startups.`
- `Identify top-performing visual content patterns from our primary competitors this week.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a research analyst, prioritizing visual relevance and trend accuracy.
- **Instruction Pattern:**
    - Focus on identifying visual patterns rather than just text-based metadata.
    - Maintain a strict filter for high-resolution and brand-compliant assets.
    - Summarize findings with actionable recommendations for the creative team.

### 2) Composio Toolset Node
Requires an active API key for your chosen media or social platform connectors. Ensure the connection scope includes "Read" and "Search" permissions for the target platforms.

### 3) Tool Availability
- **Visual Search API**: For querying image databases and trend repositories.
- **Social Media Connectors**: For monitoring platform-specific visual performance.
- **Asset Management Tools**: For saving and organizing discovered media files.
