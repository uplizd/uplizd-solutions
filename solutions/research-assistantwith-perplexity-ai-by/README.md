## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the solution template.
2. Select your preferred workspace and click "Import Flow."
3. Authenticate your Perplexity AI and Composio connections in the settings.
4. Ensure nodes are correctly wired: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives your research query or specific topic.
*   **Agent**: Processes the intent and triggers the necessary search operations.
*   **Composio Toolset**: Executes the Perplexity AI search and data retrieval.
*   **Chat Output**: Delivers the synthesized research report to the user.

### 3) Run the Flow
Open the Playground and test with these prompts:
- `Research the latest trends in generative AI for enterprise sales and provide a summary with citations.`
- `Compare the top 3 competitors in the CRM space based on recent 2024 product updates.`
- `Find recent news regarding [Company Name] and summarize their strategic focus for the upcoming quarter.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a research analyst, prioritizing accuracy and source attribution.
- Maintain a neutral, professional tone for all summaries.
- Always prioritize the most recent search results over dated information.
- Structure outputs with clear headings and bulleted summaries for readability.

### 2) Composio Toolset Node
- Requires a valid Perplexity AI API key configured within the Composio dashboard.
- Ensure the connection scope allows for web searching and data retrieval permissions.

### 3) Tool Availability
- **Perplexity Search**: Core engine for real-time web queries.
- **Web Scraper**: Optional tool for extracting specific text from identified URLs.
- **Data Formatter**: Utility for converting raw search results into structured Markdown reports.
