## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the solution template.
2. Select your preferred workspace to import the workflow.
3. Authenticate your required integrations via the Composio Toolset.
4. Ensure nodes are correctly connected: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives venue specifications, attendee counts, and event constraints.
- **Agent**: Processes spatial logic and generates layout recommendations.
- **Composio Toolset**: Executes API calls to mapping and logistics platforms.
- **Chat Output**: Delivers the finalized venue plan and logistics summary.

### 3) Run the Flow
Use the Playground to test your configurations with these prompts:
- `Create a floor plan for a 500-person conference with a main stage and 4 breakout rooms.`
- `Optimize the layout for a trade show booth arrangement in a 10,000 sq ft hall.`
- `List the logistics requirements and load-in zones for a gala event at the Grand Ballroom.`
