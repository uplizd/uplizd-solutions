## Summary
The Customer Appreciation Agent is an automated workflow designed to strengthen client loyalty by triggering personalized, handwritten-style thank you notes following key business milestones. By integrating CRM data with physical mailing services via the Composio Toolset, this solution eliminates the manual effort of relationship management, ensuring every high-value customer receives timely, meaningful recognition that drives retention and brand advocacy.

## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your preferred workspace to import the workflow.
3. Authenticate your CRM and mailing service connections within the Composio Toolset.
4. Ensure nodes are correctly mapped and all API credentials are saved before activating.

### 2) Setup the Nodes
*   **Chat Input**: Receives the CRM event trigger or manual request for appreciation.
*   **Agent**: Processes the client context and drafts a personalized message.
*   **Composio Toolset**: Executes the API calls to the mailing service and updates the CRM.
*   **Chat Output**: Confirms the successful dispatch of the note and logs the activity.

### 3) Run the Flow
Use the Playground to test the agent with prompts such as:
*   `"Send a thank you note to John Doe for renewing his annual subscription."`
*   `"Draft and mail a congratulatory note to the account owner for hitting their 6-month milestone."`
*   `"Verify the address for Acme Corp and send a personalized appreciation card."`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a professional relationship manager, focusing on warmth and authenticity.
*   Maintain a professional yet personal tone in all drafted messages.
*   Reference specific client achievements or milestones provided in the input.
*   Keep messages concise, typically under 150 words for physical note compatibility.

### 2) Composio Toolset Node
*   **API Key**: Provide your mailing service and CRM API keys in the Composio dashboard.
*   **Connection Scope**: Ensure the agent has read access to CRM contacts and write access to the mailing service provider.

### 3) Tool Availability
*   **CRM Connector**: For retrieving client contact details and logging activity history.
*   **Mailing Service API**: For creating and dispatching physical handwritten-style notes.
*   **Address Validator**: For confirming valid shipping locations before request submission.
