## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Team Collaboration Enhancer template from the solutions library.
3. Authenticate your required communication and management tools via the Composio connection portal.
4. Ensure nodes are correctly connected: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives team updates or coaching requests from your connected platforms.
- **Agent**: Processes instructions to determine the appropriate action or coaching assignment.
- **Composio Toolset**: Executes the necessary API calls to update records or send notifications.
- **Chat Output**: Delivers a confirmation or summary of the completed action to the user.

### 3) Run the Flow
Use the Playground to test the workflow with these example prompts:
- `Identify team members who haven't received coaching this month and draft a follow-up schedule.`
- `Summarize the latest team communication updates and highlight any urgent action items.`
- `Assign a new coaching task to the sales team lead based on the latest performance report.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the central intelligence for routing and decision-making.
- Focus on identifying coaching triggers from raw communication data.
- Maintain a professional and supportive tone for all team-related feedback.
- Prioritize tasks based on the urgency and impact defined in the system instructions.

### 2) Composio Toolset Node
- Provide your unique API key to enable secure access to your collaboration platforms.
- Configure the connection scope to allow the agent to read and write to your team management tools.

### 3) Tool Availability
- **Communication APIs**: For sending updates and notifications.
- **Task Management Tools**: For creating and assigning coaching tasks.
- **Reporting Connectors**: For fetching performance data to inform coaching decisions.
