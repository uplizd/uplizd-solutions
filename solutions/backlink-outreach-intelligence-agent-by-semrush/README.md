## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Backlink Outreach Intelligence Agent template from the marketplace.
3. Authenticate your Semrush account within the Composio Toolset node.
4. Ensure nodes are correctly connected: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives your target keywords or competitor URLs.
- **Agent**: Processes the request and determines the necessary research steps.
- **Composio Toolset**: Executes API calls to Semrush to fetch backlink and domain data.
- **Chat Output**: Returns a structured list of prospects with actionable research insights.

### 3) Run the Flow
Use the Playground to test the agent with prompts such as:
- `Find 10 high-authority backlink opportunities for my domain in the sustainable fashion niche.`
- `Research the backlink profile of [competitor-url.com] and suggest 5 sites for outreach.`
- `Provide a summary of the top 3 backlink prospects for my new article on AI automation.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as an SEO strategist.
- Use a model capable of complex reasoning (e.g., GPT-4o or Claude 3.5 Sonnet).
- Instruction: "You are an expert SEO strategist. Use the provided tools to find high-quality backlink prospects and summarize why each site is a good fit for outreach."
- Ensure the agent maintains a professional, data-driven tone.

### 2) Composio Toolset Node
- Provide your Semrush API key in the connection settings.
- Ensure the scope includes read access to backlink analytics and domain overview endpoints.

### 3) Tool Availability
- **Semrush Domain Analytics**: For retrieving domain authority and traffic metrics.
- **Semrush Backlink Explorer**: For identifying specific link-building opportunities.
- **Web Search/Scraping**: For gathering contact information and recent site content.
