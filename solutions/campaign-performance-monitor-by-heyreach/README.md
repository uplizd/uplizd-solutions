## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the solution in the builder.
2. Connect your HeyReach and CRM accounts via the integration settings.
3. Configure the trigger interval to match your reporting cadence (e.g., hourly or daily).
4. Ensure nodes are correctly mapped and all API credentials are verified before deployment.

### 2) Setup the Nodes
- **Chat Input**: Receives the request to initiate a performance scan or generate a report.
- **Agent**: Analyzes the raw campaign data and synthesizes actionable recommendations.
- **Composio Toolset**: Executes data retrieval from HeyReach and updates records in your CRM.
- **Chat Output**: Delivers the final performance summary and optimization suggestions to the user.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
- `Analyze the performance of my latest outreach campaign and identify the top 3 bottlenecks.`
- `Generate a summary report for all active HeyReach campaigns from the last 7 days.`
- `Which outreach sequences are currently underperforming and need a copy adjustment?`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical engine, interpreting complex campaign data into human-readable insights.
- Instruct the agent to prioritize high-impact metrics like conversion and reply rates.
- Use a structured output format to ensure the summary is easy to read.
- Maintain a professional, data-focused tone for all generated insights.

### 2) Composio Toolset Node
- Provide your HeyReach API key to enable secure data extraction.
- Define the connection scope to allow the agent to read campaign logs and write updates to your CRM.

### 3) Tool Availability
- **HeyReach Connector**: Fetches campaign status, engagement metrics, and lead interaction logs.
- **CRM Integration**: Syncs performance data and updates lead statuses based on outreach outcomes.
- **Data Processor**: Formats raw API responses into structured JSON for the Agent to analyze.
