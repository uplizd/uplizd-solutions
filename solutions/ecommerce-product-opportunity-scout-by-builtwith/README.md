## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the E-commerce Product Opportunity Scout template from the marketplace.
3. Connect your BuiltWith API credentials within the integration settings.
4. Ensure nodes are correctly linked from **Chat Input** to **Agent**, **Composio Toolset**, and finally **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives your specific market research queries or industry focus areas.
- **Agent**: Orchestrates the analysis, interpreting user intent and directing the toolset.
- **Composio Toolset**: Executes data retrieval tasks via the BuiltWith API to fetch technology usage statistics.
- **Chat Output**: Delivers a structured report detailing identified opportunities and market insights.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
- `Analyze the current adoption trends for headless e-commerce platforms in the fashion retail sector.`
- `What are the top 5 emerging technologies being adopted by mid-market e-commerce stores this quarter?`
- `Identify potential product gaps for Shopify-based stores in the home decor category based on recent tech stack shifts.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your lead market analyst, synthesizing technical data into business strategy.
- **Role:** Expert E-commerce Market Researcher.
- **Instruction Pattern:** 
    - Focus on identifying outliers and growth signals in technology adoption.
    - Prioritize actionable insights over raw data dumps.
    - Maintain a professional, strategic tone suitable for executive reporting.

### 2) Composio Toolset Node
- **API Key:** Ensure your BuiltWith API key is securely stored in the environment variables.
- **Connection Scope:** Grant the agent read-only access to technology lookup and trend analysis endpoints.

### 3) Tool Availability
- **Technology Lookup**: Retrieve the full tech stack for specific domains or categories.
- **Trend Analysis**: Aggregate usage data over time to identify growth or decline in technology popularity.
- **Domain Filtering**: Narrow results by industry, revenue tier, or geographic location.
