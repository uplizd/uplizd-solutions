## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to navigate to the solution page.
2. Select "Import Workflow" to add the Carbon Market Intelligence Agent to your workspace.
3. Authenticate your required web scraping and notification services via the Composio Toolset.
4. Ensure nodes are correctly connected: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives your specific market queries or monitoring parameters.
- **Agent**: Processes market data and applies analytical logic to identify trends.
- **Composio Toolset**: Executes web scraping and data retrieval tasks across configured sources.
- **Chat Output**: Delivers the final analysis, alerts, or reports directly to your dashboard.

### 3) Run the Flow
Use the Playground to test your agent with these example prompts:
- `Analyze the current price trends for nature-based carbon removal credits over the last 30 days.`
- `Alert me if any new carbon removal projects are listed in the European market today.`
- `Summarize the impact of recent regulatory changes on carbon credit supply for Q3.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical brain, interpreting scraped data and formatting it for human readability.
- Use a high-reasoning model (e.g., GPT-4o or Claude 3.5 Sonnet) for accurate data synthesis.
- Instruct the agent to prioritize data recency and source credibility.
- Configure the system prompt to maintain a professional, analytical tone for ESG reporting.

### 2) Composio Toolset Node
- Provide your API keys for the selected web scraping and data integration providers.
- Set the connection scope to allow the agent to read public market data and perform search queries.

### 3) Tool Availability
- **Web Search/Scraper**: Access to real-time market data and news.
- **Data Parser**: Capability to extract structured information from unstructured web content.
- **Notification Service**: Ability to push alerts to Slack, Email, or internal CRM systems.
