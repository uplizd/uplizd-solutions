## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the template.
2. Select your workspace and click "Import Flow."
3. Connect your required API credentials within the integration settings.
4. Ensure nodes are correctly mapped and all required environment variables are populated.

### 2) Setup the Nodes
- **Chat Input**: Receives user queries regarding portfolio status or specific asset performance.
- **Agent**: Processes natural language requests and orchestrates the logic for data retrieval.
- **Composio Toolset**: Executes secure API calls to your portfolio and analytics platforms to fetch real-time data.
- **Chat Output**: Delivers formatted reports, alerts, or actionable insights back to the user.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
- `Generate a health report for my 'Wedding' gallery for the last 30 days.`
- `Are there any broken links or missing alt-text fields in my current portfolio?`
- `Which three projects have the highest engagement rate this week?`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your portfolio analyst, translating complex data into human-readable insights.
- Instruct the agent to prioritize high-impact metrics like conversion and engagement.
- Maintain a professional, analytical tone for all generated reports.
- Ensure the agent cross-references multiple data sources before flagging an asset as "underperforming."

### 2) Composio Toolset Node
- Provide your API key for the relevant portfolio management or analytics platform.
- Configure the connection scope to allow read-access to asset metadata and visitor analytics.

### 3) Tool Availability
- **Asset Management API**: For scanning, updating, and verifying image links.
- **Analytics Connector**: For pulling traffic, bounce rates, and engagement metrics.
- **Notification Service**: For sending health alerts to your preferred communication channel.
