## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the solution template.
2. Select your preferred workspace to import the workflow.
3. Authenticate your required connectors via the Composio dashboard.
4. Ensure nodes are correctly mapped and all API keys are active in the configuration panel.

### 2) Setup the Nodes
- **Chat Input**: Receives the specific ESG reporting query or regulatory focus area.
- **Agent**: Processes the request, performs web research, and synthesizes compliance data.
- **Composio Toolset**: Executes the scraping and data retrieval tasks across targeted web sources.
- **Chat Output**: Delivers the finalized report or compliance summary to the user.

### 3) Run the Flow
Use the Playground to test the agent with these prompts:
- `Check for the latest updates to the EU Corporate Sustainability Reporting Directive.`
- `Scrape the latest environmental impact guidelines from the EPA website.`
- `Summarize our current compliance status based on the latest sustainability data.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a specialized compliance analyst.
- Use a high-reasoning model (e.g., GPT-4o or Claude 3.5 Sonnet) for accurate data synthesis.
- Instruct the agent to prioritize official government and regulatory body sources.
- Configure the system prompt to maintain a formal, audit-compliant tone in all outputs.

### 2) Composio Toolset Node
- Provide a valid API key for the scraping and web-access tools.
- Set the connection scope to allow read-only access to relevant regulatory databases and internal document repositories.

### 3) Tool Availability
- **Web Scraper**: For gathering real-time regulatory updates.
- **Document Parser**: For extracting metrics from PDF and CSV reports.
- **Notification Service**: For alerting the team to compliance changes.
