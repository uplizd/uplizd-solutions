## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and select the Instagram Copywriter solution.
2. Click "Import" to add the workflow to your workspace.
3. Connect your preferred LLM and social media integration credentials.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the topic, target audience, and brand tone from the user.
- **Agent**: Processes the input to draft creative copy and visual prompts.
- **Composio Toolset**: Executes social media API actions for scheduling or publishing.
- **Chat Output**: Displays the final caption, hashtag set, and image prompt for review.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Write an engaging Instagram caption for a new sustainable coffee launch, include 5 relevant hashtags.`
- `Create a professional caption for a B2B software update announcement and suggest a prompt for a minimalist graphic.`
- `Draft a fun, casual caption for a team culture post featuring our new office space.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your creative lead, ensuring content is both engaging and on-brand.
- Set the system prompt to define your specific brand voice (e.g., "Professional yet witty").
- Enable "Creative Mode" to allow the model to suggest unique angles for captions.
- Configure the output format to separate captions from hashtag blocks for easier copying.

### 2) Composio Toolset Node
- Provide your API key to enable secure communication with your social media platforms.
- Define the connection scope to allow the agent to read account analytics or post directly to your feed.

### 3) Tool Availability
- **Content Generator**: Core engine for text and prompt creation.
- **Social Media Connector**: Manages API authentication and publishing endpoints.
- **Trend Analyzer**: Provides real-time hashtag and engagement data.
