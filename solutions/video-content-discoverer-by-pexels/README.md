## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the solution template.
2. Select your workspace and import the workflow into your dashboard.
3. Connect your Pexels API credentials within the Composio Toolset configuration.
4. Ensure nodes are correctly linked: **Chat Input** → **Agent** → **Composio Toolset** → **Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives your natural language request for video content.
*   **Agent**: Processes the request and determines the optimal search terms for the Pexels API.
*   **Composio Toolset**: Executes the search query against the Pexels platform to retrieve asset metadata and links.
*   **Chat Output**: Delivers the curated list of video assets directly to your chat interface.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Find 5 high-quality 4K videos of modern office workspaces for a corporate presentation.`
- `Search for trending nature and landscape videos suitable for a travel marketing campaign.`
- `Get me some cinematic footage of urban night life for a social media ad series.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the intelligent bridge between your creative requirements and the Pexels search engine.
- Use a model capable of high-level reasoning to parse abstract creative briefs.
- Instruct the agent to prioritize high-resolution and high-rated assets.
- Configure the agent to provide direct links and descriptions for each retrieved video.

### 2) Composio Toolset Node
- Provide your Pexels API key to authorize the connection.
- Set the connection scope to allow read-only access for searching and retrieving video metadata.

### 3) Tool Availability
- **Pexels Search**: Capability to query the video database using keywords, orientation, and size filters.
- **Asset Metadata Retrieval**: Ability to extract direct download URLs, author information, and video duration.
