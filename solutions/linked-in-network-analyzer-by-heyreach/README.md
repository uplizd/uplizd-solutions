## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and select the LinkedIn Network Analyzer solution.
2. Click "Import" to add the workflow to your workspace.
3. Authenticate your LinkedIn account via the Composio connection prompt.
4. Ensure nodes are correctly connected: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives your specific filtering criteria or target industry keywords.
- **Agent**: Analyzes the provided LinkedIn data against your defined persona and outreach goals.
- **Composio Toolset**: Executes the API calls to fetch and process LinkedIn connection data.
- **Chat Output**: Delivers a structured report of prioritized leads and recommended next steps.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Analyze my current LinkedIn connections and identify the top 10 prospects in the SaaS industry.`
- `Find connections who are currently in Director-level roles and have been active in the last 30 days.`
- `Generate a personalized outreach message for my top 5 priority connections identified in the last audit.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as your personal sales assistant, interpreting network data to provide strategic recommendations.
- Focus on identifying high-intent signals within profile descriptions.
- Maintain a professional, consultative tone for all outreach suggestions.
- Prioritize accuracy by cross-referencing connection activity with your target market criteria.

### 2) Composio Toolset Node
- **API Key**: Ensure your LinkedIn API key is active within the Composio dashboard.
- **Connection Scope**: Grant read-only access to your network and profile activity to ensure secure data retrieval.

### 3) Tool Availability
- **Profile Fetcher**: Retrieves detailed metadata for specific connections.
- **Activity Monitor**: Tracks recent posts and engagement patterns.
- **Search & Filter**: Enables complex queries across your entire network database.
