## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the solution template.
2. Select your workspace and import the workflow into your dashboard.
3. Authenticate your YouTube credentials within the Composio connection settings.
4. Ensure nodes are correctly mapped from Chat Input to Chat Output before initiating the first run.

### 2) Setup the Nodes
- **Chat Input**: Receives your research query or niche topic.
- **Agent**: Processes the request and determines the necessary search parameters.
- **Composio Toolset**: Executes API calls to fetch YouTube data and trend metrics.
- **Chat Output**: Delivers the synthesized research report and content recommendations.

### 3) Run the Flow
Open the Playground and test the agent with these prompts:
- `Analyze the top 5 trending topics in the "AI productivity" niche for this month.`
- `What are the most common viewer questions in the comments of these 3 competitor videos?`
- `Suggest 5 video title ideas based on current high-performing search queries for sustainable tech.`

## Configuration
### 1) Language Model (Agent Node)
The Agent node acts as the brain of the research process.
- **Instruction Pattern:**
    - Act as a professional YouTube growth consultant.
    - Prioritize data-backed insights over generic suggestions.
    - Format output as a structured report with actionable next steps.

### 2) Composio Toolset Node
- Requires a valid YouTube API key and OAuth connection.
- Ensure the connection scope includes read-only access to public video data and search analytics.

### 3) Tool Availability
- **YouTube Search API**: For finding trending videos and keywords.
- **Video Metadata Extractor**: For pulling titles, descriptions, and view counts.
- **Comment Analysis Tool**: For sentiment and feedback aggregation.
