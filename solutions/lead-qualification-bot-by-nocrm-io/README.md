## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" button above to open the template.
2. Connect your noCRM.io account within the Composio Toolset node.
3. Configure your specific qualification criteria in the Agent node instructions.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives raw lead data from your web forms or email parser.
*   **Agent**: Evaluates the lead data against your defined scoring model and business logic.
*   **Composio Toolset**: Executes the API calls to create or update records in noCRM.io.
*   **Chat Output**: Returns a summary of the qualification result and the action taken in the CRM.

### 3) Run the Flow
Test the workflow in the Uplizd Playground:
*   `Qualify this lead: John Doe from Acme Corp, interested in enterprise pricing.`
*   `Check if this lead meets our B2B criteria and add to noCRM.io if qualified.`
*   `Update the lead score for lead ID 12345 based on the latest interaction.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as your digital sales assistant, interpreting lead data and applying business logic.
*   Use a model with strong reasoning capabilities (e.g., GPT-4o or Claude 3.5 Sonnet).
*   Define clear "Qualification Rules" in the system prompt to ensure consistent scoring.
*   Instruct the agent to output structured data for the Composio Toolset to process.

### 2) Composio Toolset Node
*   **API Key**: Provide your noCRM.io API key within the Composio configuration.
*   **Connection Scope**: Ensure the agent has read/write permissions for lead objects and custom fields.

### 3) Tool Availability
*   `nocrm_create_lead`: Create new entries from qualified inbound traffic.
*   `nocrm_update_lead`: Modify existing lead statuses and scores.
*   `nocrm_search_lead`: Prevent duplicates by checking for existing records before creation.
