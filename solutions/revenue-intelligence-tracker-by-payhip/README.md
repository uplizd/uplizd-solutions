## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your workspace and import the Revenue Intelligence Tracker flow.
3. Authenticate your Payhip and CRM connectors within the Composio integration settings.
4. Ensure nodes are correctly connected: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives the trigger or manual request to analyze revenue data.
*   **Agent**: Processes the request using the configured LLM to interpret financial queries.
*   **Composio Toolset**: Executes the necessary API calls to fetch and structure Payhip data.
*   **Chat Output**: Delivers the formatted revenue insights and analysis back to the user.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
* `Analyze the total revenue generated by Payhip for the last 7 days.`
* `List the top 5 products by revenue and identify any significant trends.`
* `Compare this month's revenue performance against the previous month.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical engine, translating natural language queries into data-driven insights.
*   Use a model with strong reasoning capabilities (e.g., GPT-4o or Claude 3.5 Sonnet).
*   Provide clear instructions on how to handle missing data or zero-revenue periods.
*   Ensure the system prompt emphasizes accuracy in financial calculations.

### 2) Composio Toolset Node
*   **API Key**: Provide your Payhip API credentials within the Composio dashboard.
*   **Connection Scope**: Ensure the toolset has read-access to transaction and product endpoints.

### 3) Tool Availability
*   **Transaction Fetcher**: Retrieves raw sales data from Payhip.
*   **Data Formatter**: Normalizes currency and date fields for consistent analysis.
*   **Report Generator**: Compiles processed data into human-readable summaries.
