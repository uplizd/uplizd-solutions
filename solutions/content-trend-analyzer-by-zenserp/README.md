## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Content Trend Analyzer template from the marketplace.
3. Connect your Zenserp API credentials within the integration settings.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives your target industry or niche keywords.
- **Agent**: Processes search data and applies content strategy logic.
- **Composio Toolset**: Executes Zenserp queries to fetch live search results.
- **Chat Output**: Delivers a structured report of trending opportunities and content suggestions.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
- `Analyze the top 5 trending search queries for 'sustainable fashion' this week.`
- `Find high-intent keywords in the 'remote work software' niche that have low competition.`
- `Generate a content calendar based on current search trends for 'AI marketing tools'.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a market analyst, synthesizing raw search data into actionable insights.
- Instruct the agent to prioritize high-volume, low-competition keywords.
- Set the tone to be analytical, concise, and focused on growth metrics.
- Ensure the agent provides a clear "Why this matters" summary for every identified trend.

### 2) Composio Toolset Node
- Provide your Zenserp API key in the toolset configuration.
- Set the connection scope to allow read-only access to search trend and keyword data.

### 3) Tool Availability
- **Zenserp Search API**: For real-time keyword volume and ranking data.
- **Trend Aggregator**: For historical analysis and velocity calculations.
- **Content Generator**: For drafting titles and outlines based on identified trends.
