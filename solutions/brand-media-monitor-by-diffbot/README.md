## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the solution template.
2. Select your preferred workspace to initialize the workflow.
3. Authenticate the required Diffbot and CRM integrations within the Composio Toolset.
4. Ensure nodes are correctly connected from Chat Input to Agent, and Agent to Composio Toolset.

### 2) Setup the Nodes
- **Chat Input**: Receives your brand keywords or specific URLs to monitor.
- **Agent**: Processes search queries and performs sentiment analysis on the retrieved data.
- **Composio Toolset**: Executes the Diffbot web extraction and data formatting tasks.
- **Chat Output**: Delivers a summarized report of findings and actionable insights.

### 3) Run the Flow
Use the Playground to test your monitor with prompts like:
- `Find all recent mentions of [Brand Name] and summarize the overall sentiment.`
- `Monitor [Competitor Name] for new product announcements this week.`
- `Create a summary report of industry trends related to [Topic] from the last 24 hours.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical brain, filtering raw web data into meaningful summaries.
- Use a high-reasoning model (e.g., GPT-4o) for accurate sentiment classification.
- Configure system instructions to prioritize recent data and filter out irrelevant noise.
- Set output formatting to JSON or Markdown for easy integration with downstream tools.

### 2) Composio Toolset Node
- Provide your Diffbot API key to enable high-fidelity web extraction.
- Define the connection scope to include read-only access for web search and extraction tools.

### 3) Tool Availability
- **Web Extraction**: Parses article text, metadata, and sentiment signals.
- **Search Query Engine**: Executes targeted queries across indexed web content.
- **Data Formatter**: Converts unstructured HTML/text into structured reports.
