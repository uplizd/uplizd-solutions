## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd Solutions library and select the Carbon Offset Advisory Agent.
2. Click "Import" to add the workflow template to your workspace.
3. Connect your required API credentials within the integration settings.
4. Ensure nodes are correctly mapped and the Agent has access to the necessary Composio Toolset.

### 2) Setup the Nodes
- **Chat Input**: Receives your specific sustainability queries or procurement requests.
- **Agent**: Processes environmental data and evaluates offset options against your criteria.
- **Composio Toolset**: Executes data retrieval and purchasing commands across connected platforms.
- **Chat Output**: Delivers actionable advice, project summaries, or confirmation of completed transactions.

### 3) Run the Flow
Use the Playground to test the agent with the following prompts:
- `Find high-quality carbon removal projects in the forestry sector with Gold Standard certification.`
- `Calculate the offset volume needed for our Q3 operational emissions and suggest three purchasing options.`
- `Provide a status update on our recent carbon credit acquisitions and generate a summary report.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a specialized sustainability consultant.
- Use a model with strong reasoning capabilities to interpret complex environmental data.
- Instruct the agent to prioritize verified, high-impact projects over lower-cost alternatives.
- Maintain a professional, data-driven tone suitable for corporate ESG reporting.

### 2) Composio Toolset Node
- Provide the necessary API keys for your preferred carbon market or environmental data platforms.
- Set the connection scope to allow the agent to read project data and initiate purchase requests.

### 3) Tool Availability
- **Marketplace Search**: Capability to query and filter available carbon removal credits.
- **Project Verification**: Access to databases for validating project certifications and impact scores.
- **Transaction Manager**: Ability to execute and track purchase orders for selected offsets.
