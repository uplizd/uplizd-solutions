## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge to open the solution template.
2. Select your preferred workspace to import the workflow.
3. Authenticate your required data sources within the Composio Toolset.
4. Ensure nodes are correctly connected from Chat Input to Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives your specific market research queries or location parameters.
- **Agent**: Analyzes the input and determines the necessary data retrieval steps.
- **Composio Toolset**: Executes API calls to fetch and map competitor location data.
- **Chat Output**: Delivers the structured mapping results and strategic insights back to the user.

### 3) Run the Flow
Use the Playground to test your mapping logic with these prompts:
- `Map all competitors within a 50-mile radius of the downtown Chicago area.`
- `Identify regions in the Northeast with high market demand but low competitor density.`
- `Generate a summary report of competitor expansion trends for the last quarter.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the analytical engine for processing market intelligence.
- Use a model capable of complex reasoning and structured data output.
- Instruct the agent to prioritize accuracy in geographic mapping and competitor identification.
- Configure the system prompt to maintain a professional, analytical tone for all output.

### 2) Composio Toolset Node
- Provide the necessary API keys for your market data providers.
- Ensure the connection scope includes read access to location and business intelligence endpoints.

### 3) Tool Availability
- **Geographic Data API**: For fetching location coordinates and regional boundaries.
- **Market Intelligence Connector**: For accessing competitor performance and density metrics.
- **Data Transformation Utilities**: For standardizing and mapping raw data into the required schema.
