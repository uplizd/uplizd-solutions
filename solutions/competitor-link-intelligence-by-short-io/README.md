## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Competitor Link Intelligence template file.
3. Authenticate your Short.io or relevant URL management credentials via the Composio connection manager.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Accepts the target URL or competitor domain to be analyzed.
- **Agent**: Orchestrates the logic to query link data and synthesize competitive insights.
- **Composio Toolset**: Executes API calls to resolve redirects and fetch link metadata.
- **Chat Output**: Displays the decoded destination, campaign context, and strategic recommendations.

### 3) Run the Flow
Use the Playground to test the workflow with these prompts:
- `Analyze the destination of this competitor link: [URL]`
- `What are the recent campaign patterns for [Competitor Domain]?`
- `Compare the landing page structure of these two competitor links and summarize the differences.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a marketing strategist, interpreting technical link data into business intelligence.
- **Instruction Pattern:**
  - "You are an expert competitive intelligence analyst specializing in digital marketing."
  - "Always resolve the final destination URL before providing strategic analysis."
  - "Focus on identifying the intent behind the link (e.g., lead gen, content distribution, or sales)."

### 2) Composio Toolset Node
- **API Key:** Ensure your Short.io or link management provider API key is active in the Composio dashboard.
- **Connection Scope:** Grant read-only access to link analytics and redirect resolution tools.

### 3) Tool Availability
- **URL Resolver**: Capability to follow redirects and extract final destination metadata.
- **Link Analytics Fetcher**: Access to historical click data and geographic distribution (if available).
- **Competitor Database Query**: Ability to cross-reference link patterns against stored historical data.
