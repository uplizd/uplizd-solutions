## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Content Vocabulary Enhancer template from the solutions library.
3. Connect your preferred Dictionary API credentials within the Composio Toolset node.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the raw text draft from the user.
- **Agent**: Analyzes the text for vocabulary improvements and triggers the dictionary tool.
- **Composio Toolset**: Executes the API calls to fetch definitions and synonyms.
- **Chat Output**: Returns the enhanced text with suggested edits and definitions.

### 3) Run the Flow
Use the Playground to test your workflow with these example prompts:
- `Enhance the vocabulary of this paragraph: "The company made a big change to their strategy to get more sales."`
- `Find three sophisticated synonyms for "important" that fit a professional business context.`
- `Provide a clear, simple definition for "synergy" that I can use in an internal newsletter.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a linguistic editor.
- Use a model with strong reasoning capabilities (e.g., GPT-4o or Claude 3.5 Sonnet).
- Instruction pattern: "You are a professional editor. Identify complex or weak words in the input. Use the provided dictionary tool to find precise definitions and synonyms. Rewrite the input to improve clarity and impact."

### 2) Composio Toolset Node
- Provide your valid API key for the chosen dictionary service.
- Set the connection scope to allow read-only access to linguistic databases.

### 3) Tool Availability
- **Dictionary Lookup**: Fetches definitions for specific words.
- **Thesaurus Query**: Retrieves context-aware synonyms.
- **Readability Analyzer**: Calculates text complexity metrics.
