## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to open the solution template.
2. Select your workspace and click "Import Flow" to initialize the builder.
3. Authenticate your required productivity integrations within the Composio dashboard.
4. Ensure nodes are correctly connected from Chat Input to the Agent and finally to the Chat Output.

### 2) Setup the Nodes
*   **Chat Input**: Receives your habit-related queries or manual activity logs.
*   **Agent**: Processes behavioral data and generates personalized habit strategies.
*   **Composio Toolset**: Executes data retrieval and updates across your connected productivity apps.
*   **Chat Output**: Delivers actionable insights, progress reports, and personalized habit prompts.

### 3) Run the Flow
Open the Playground and test the workflow with these prompts:
* `Analyze my habit consistency over the last 7 days and suggest one improvement.`
* `What are the primary blockers preventing me from completing my morning reading habit?`
* `Sync my latest activity data and update my progress report for the current goal.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as a behavioral coach, synthesizing data into clear, encouraging, and actionable advice.
*   Adopt a supportive, objective, and analytical persona.
*   Prioritize data-backed insights over generic motivational content.
*   Maintain a consistent tone that focuses on incremental progress and habit hygiene.

### 2) Composio Toolset Node
Requires a valid API key and authorized access to your productivity platforms (e.g., Calendar, Task Managers, or Note-taking apps). Ensure the connection scope includes read/write permissions for relevant activity data.

### 3) Tool Availability
*   **Data Aggregator**: Fetches logs from connected productivity tools.
*   **Pattern Analyzer**: Performs statistical analysis on activity timestamps and completion status.
*   **Notification Dispatcher**: Sends reminders based on configured habit triggers.
