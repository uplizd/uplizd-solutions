## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" badge above to navigate to the solution page.
2. Select "Import Workflow" to add the Dividend Income Optimizer to your workspace.
3. Connect your preferred financial data providers within the Composio Toolset.
4. Ensure nodes are correctly mapped from **Chat Input** to **Agent** to **Composio Toolset** to **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives your portfolio ticker symbols or specific financial queries.
- **Agent**: Processes financial logic and interprets market data requests.
- **Composio Toolset**: Executes secure API calls to fetch real-time dividend and market data.
- **Chat Output**: Delivers formatted income forecasts and portfolio insights back to the user.

### 3) Run the Flow
Use the Uplizd Playground to test your workflow with these prompts:
- `Calculate the projected annual dividend income for my current portfolio.`
- `Identify which assets in my portfolio have the highest dividend yield growth over the last 3 years.`
- `Create a summary report of all upcoming dividend payouts for the next 30 days.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as your financial research assistant, translating natural language queries into structured data requests.
- Use a high-reasoning model to ensure accurate calculation of yield projections.
- Instruct the agent to prioritize verified financial data sources.
- Configure the agent to provide clear, concise summaries of complex financial datasets.

### 2) Composio Toolset Node
- Provide your API keys for the relevant financial data providers (e.g., Nasdaq Data Link).
- Set the connection scope to allow read-only access to market data and ticker information.

### 3) Tool Availability
- **Market Data Fetcher**: Retrieves real-time stock prices and dividend history.
- **Yield Calculator**: Performs mathematical operations on payout ratios and share counts.
- **Report Generator**: Formats raw data into readable tables and summaries.
