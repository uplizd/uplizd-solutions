## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" button above to open the template in your workspace.
2. Connect your required analytics and marketing tool accounts via the Composio Toolset.
3. Configure the trigger settings to match your preferred data polling frequency.
4. Ensure nodes are correctly mapped from **Chat Input** to **Agent** to **Composio Toolset** to **Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives the trigger signal or manual request for performance analysis.
- **Agent**: Processes performance data and determines the optimal campaign adjustment.
- **Composio Toolset**: Executes the API calls to update campaign settings or fetch analytics data.
- **Chat Output**: Delivers a summary report of the optimizations performed and current campaign status.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
`Analyze the performance of the latest social proof campaign and suggest three variants.`
`Update the landing page testimonials based on the top 5% conversion drivers from last week.`
`Generate a summary report of all automated optimizations performed in the last 24 hours.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a data analyst and campaign strategist.
- Use a model capable of logical reasoning and data interpretation.
- Instruction: "Analyze the provided performance metrics and identify the most effective social proof elements."
- Instruction: "Maintain a neutral, data-backed tone when reporting on campaign performance and suggested changes."

### 2) Composio Toolset Node
- Provide your API keys for the relevant analytics and marketing platforms.
- Set the connection scope to allow read access to performance metrics and write access to campaign assets.

### 3) Tool Availability
- **Analytics Connectors**: Fetching campaign performance and conversion data.
- **Marketing Platform APIs**: Updating social proof assets and campaign parameters.
- **Notification Services**: Alerting the team to significant performance shifts.
