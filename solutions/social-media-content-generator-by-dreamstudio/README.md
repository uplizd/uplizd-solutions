## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" button to open the solution in the builder.
2. Connect your DreamStudio and relevant social platform API keys via the Composio Toolset.
3. Configure the Agent node with your preferred brand voice instructions.
4. Ensure nodes are correctly linked from **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
*   **Chat Input**: Receives the content brief, theme, and platform requirements from the user.
*   **Agent**: Processes the input to generate creative prompts and marketing copy.
*   **Composio Toolset**: Executes the API calls to DreamStudio to render the visual assets.
*   **Chat Output**: Delivers the final generated image URL and the accompanying social media caption.

### 3) Run the Flow
*   `Generate a professional LinkedIn post image about sustainable tech with a minimalist aesthetic.`
*   `Create a vibrant, energetic Instagram post visual for a summer sale campaign.`
*   `Design a futuristic background image for a tech webinar announcement and write a short, punchy caption.`

## Configuration
### 1) Language Model (Agent Node)
The Agent acts as the creative director, translating user intent into precise visual prompts.
*   Maintain a consistent brand voice by defining specific adjectives for the AI to use.
*   Instruct the agent to prioritize high-contrast, visually appealing compositions.
*   Ensure the agent includes platform-specific hashtags and call-to-action phrases in the output.

### 2) Composio Toolset Node
*   Requires a valid DreamStudio API key to authorize image generation requests.
*   Scope should be limited to the specific creative tools required for your content production workflow.

### 3) Tool Availability
*   **DreamStudio API**: For high-quality image rendering and style transfer.
*   **Prompt Optimization Tool**: To refine user input for better visual results.
*   **Content Formatting Tool**: To structure copy for specific social media platform requirements.
