## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the TikTok Cross-Platform Content Distributor template using the provided solution URL.
3. Connect your TikTok and target social media accounts within the Composio Toolset node.
4. Ensure nodes are correctly linked: Chat Input → Agent → Composio Toolset → Chat Output.

### 2) Setup the Nodes
- **Chat Input**: Receives the TikTok video URL or file metadata from the user.
- **Agent**: Analyzes the content and generates platform-specific captions and metadata.
- **Composio Toolset**: Executes the API calls to upload content to selected social platforms.
- **Chat Output**: Confirms successful distribution and provides links to the new posts.

### 3) Run the Flow
Use the Playground to test your distribution:
- `Distribute my latest TikTok video to YouTube Shorts and Instagram.`
- `Repurpose this TikTok video for LinkedIn and Twitter.`
- `Sync all my recent TikTok content to my connected social media accounts.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as your social media strategist, ensuring content is optimized for every platform.
- Use a high-reasoning model to ensure accurate hashtag and caption generation.
- Configure the system prompt to maintain your specific brand voice.
- Set the temperature to 0.7 for creative yet platform-appropriate captioning.

### 2) Composio Toolset Node
- Provide your API keys for the respective social media platforms in the Composio dashboard.
- Ensure the connection scope includes `write` and `upload` permissions for all target channels.

### 3) Tool Availability
- **Social Media APIs**: Connectors for YouTube, Instagram, LinkedIn, and Twitter.
- **File Processing Tools**: Utilities for video metadata extraction and format validation.
- **Notification Tools**: Alerts for successful distribution or API errors.
