## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd marketplace and select the Ad Trend Tracking Agent.
2. Click "Import" to add the workflow to your workspace.
3. Authenticate your required ad intelligence and marketing tool connections via the Composio dashboard.
4. Ensure nodes are correctly mapped and all API keys are validated in the configuration panel.

### 2) Setup the Nodes
*   **Chat Input**: Receives user queries regarding specific market segments or competitor brands.
*   **Agent**: Processes trend data and synthesizes insights based on the provided instructions.
*   **Composio Toolset**: Executes queries to fetch real-time ad performance and creative data from external platforms.
*   **Chat Output**: Delivers the final trend report or creative recommendation to the user.

### 3) Run the Flow
Use the Playground to test the agent with the following prompts:
- `Analyze the latest creative trends for fitness apparel brands on social media over the last 7 days.`
- `Compare the messaging strategy of [Competitor Name] against our current campaign performance.`
- `What are the top 3 emerging ad formats in the SaaS industry this month?`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a strategic marketing analyst. 
- Focus on identifying actionable patterns rather than just reporting raw numbers.
- Maintain a professional, data-first tone in all generated reports.
- Prioritize insights that correlate with higher conversion rates or improved ROAS.

### 2) Composio Toolset Node
- Ensure the API key for your chosen ad intelligence platform is active.
- Set the connection scope to allow read-access to ad creative libraries and performance metrics.

### 3) Tool Availability
- **Ad Intelligence API**: For fetching competitor spend and creative data.
- **Market Analytics Connector**: For accessing industry-wide trend reports.
- **Notification Service**: For pushing alerts to Slack or Email.
