# Brand Consistency Guardian (Uplizd) - Automated AI-driven image accessibility and brand voice alignment

## Summary
The Brand Consistency Guardian is an automated AI workflow designed to ensure that all image alt text across your digital assets adheres to strict brand voice guidelines and accessibility standards. By leveraging the Composio Toolset to interface with AltText.ai and your content management systems, this solution eliminates manual tagging bottlenecks, ensures regulatory compliance, and maintains a unified brand identity across every visual touchpoint.

## Quick Start
### 1) Import the Flow into Uplizd
1. Click the "Run on Uplizd" button above to open the solution template.
2. Select your workspace and click "Import Flow" to initialize the builder.
3. Connect your AltText.ai and CMS credentials within the integration settings.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives image URLs or file paths from your content pipeline.
- **Agent**: Analyzes the image and applies your brand voice instructions to the description.
- **Composio Toolset**: Executes the API calls to generate and update alt text in your destination platform.
- **Chat Output**: Confirms successful updates and provides a summary of the processed assets.

### 3) Run the Flow
Use the Playground to test your workflow with these prompts:
- `Generate alt text for the image at [URL] using our professional brand voice.`
- `Audit the last 5 images in the media library and update their alt text for accessibility.`
- `Create a descriptive, SEO-friendly alt text for the new product launch banner.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as the bridge between raw image data and your brand guidelines.
- **Instruction Pattern**:
    - "You are a brand-aware accessibility expert. Analyze the provided image and generate alt text that is descriptive, concise, and compliant with WCAG standards."
    - "Always prioritize the core subject of the image while incorporating our brand-specific terminology."
    - "If the image is decorative, provide a null alt tag; if functional, describe the action clearly."

### 2) Composio Toolset Node
- **API Key**: Ensure your AltText.ai API key is securely stored in the environment variables.
- **Connection Scope**: Grant the toolset read/write access to your image hosting platform or CMS to allow for automated metadata updates.

### 3) Tool Availability
- **Image Analysis Engine**: Processes visual input to identify objects, text, and context.
- **Metadata Update Utility**: Pushes the generated alt text to the target image object.
- **Compliance Validator**: Checks generated text against length and descriptive quality benchmarks.
