## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Workflow."
2. Import the Market Penetration Analysis template from the solution library.
3. Configure your API credentials for the BuiltWith and CRM integrations.
4. Ensure nodes are correctly connected: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
* **Chat Input**: Receives your target market criteria or domain list.
* **Agent**: Processes the request and determines which technology data points to extract.
* **Composio Toolset**: Executes the API calls to BuiltWith to fetch real-time technology data.
* **Chat Output**: Returns the analyzed market insights and enriched lead data in a readable format.

### 3) Run the Flow
Use the Playground to test your analysis:
* `Analyze the adoption of Shopify in the fashion retail sector across North America.`
* `Find 50 companies that use Salesforce and have recently added a new marketing automation tool.`
* `Compare the technology stack of our top 10 competitors and identify our unique market advantage.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a market researcher and data analyst.
* **Instruction Pattern**:
    * Act as a GTM strategist; prioritize accuracy in technology identification.
    * Use the Composio Toolset to fetch the most recent data before providing a summary.
    * Format output as a structured table or list for easy export to CRM.

### 2) Composio Toolset Node
* **API Key**: Ensure your BuiltWith API key is active and has sufficient credits.
* **Connection Scope**: Grant the agent access to read domain technology data and write to your CRM.

### 3) Tool Availability
* **BuiltWith Lookup**: Fetches comprehensive tech stack information for specific domains.
* **Domain Search**: Identifies companies based on specific technology usage patterns.
* **CRM Connector**: Pushes enriched data directly into your sales pipeline management system.
