## Quick Start
### 1) Import the Flow into Uplizd
1. Navigate to the Uplizd dashboard and select "Create New Flow."
2. Import the Trend Discovery Agent template from the solution library.
3. Connect your required API credentials within the Composio Toolset node.
4. Ensure nodes are correctly linked: **Chat Input → Agent → Composio Toolset → Chat Output**.

### 2) Setup the Nodes
- **Chat Input**: Receives your specific market search parameters or category focus.
- **Agent**: Processes market signals using your defined instruction set.
- **Composio Toolset**: Executes data retrieval and cross-platform sync tasks.
- **Chat Output**: Returns the final trend analysis and recommended actions.

### 3) Run the Flow
Use the Playground to test the agent with prompts such as:
- `Analyze emerging trends in sustainable home goods for Q3.`
- `Identify top 5 rising product categories in the electronics sector this week.`
- `Compare current search volume for [Product A] vs [Product B] and suggest a strategy.`

## Configuration
### 1) Language Model (Agent Node)
The agent acts as a market analyst. Recommended instructions:
- Focus on identifying high-velocity trends with long-term growth potential.
- Maintain an objective tone, citing data signals whenever possible.
- Prioritize actionable insights that link directly to retail or marketing operations.

### 2) Composio Toolset Node
- Provide your API key to enable secure access to market data providers.
- Define the connection scope to include only the necessary read/write permissions for your CRM and analytics tools.

### 3) Tool Availability
- **Market Data API**: For fetching real-time search and social signal data.
- **CRM Connector**: For syncing trend insights directly into your sales or product pipeline.
- **Analytics Bridge**: For cross-referencing internal sales data with external market trends.
